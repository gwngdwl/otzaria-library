<h1>יָפוּ בְעֵינַי כָּל־דְּרָכֶיךָ</h1>
יהודה הלוי
<br/>
<blockquote><p>יָפוּ בְעֵינַי כָּל־דְּרָכֶיךָ,</p></blockquote>
<blockquote><p>גַּם יָקְרוּ לִי מַהֲלָכֶיךָ,</p></blockquote>
<blockquote><p>אֵין בָּם לְאִישׁ מִכְשׁוֹל וְלֹא מוֹקֵשׁ,</p></blockquote>
<blockquote><p>כִּי אֵין פְּתַלְתֹּל בָּם וְלֹא עִקֵּשׁ,</p></blockquote>
<blockquote><p>הִתְהַלַּכְתִּי / אוֹרְחֹתֶיךָ,</p></blockquote>
<blockquote><p>כִי נִמְשַׁכְתִּי / בַּעְבֹתֶיךָ.</p></blockquote>
<br/>
<blockquote><p>הִנֵּה בְךָ שִׂבְרִי וּמַאְוַיָּי,</p></blockquote>
<blockquote><p>אַתְּ צוּר לְבָבִי, אַתְּ מְקוֹר חַיָּי,</p></blockquote>
<blockquote><p>עֵינִי הֲדָרְךָ נִכְסְפָה לִרְאוֹת</p></blockquote>
<blockquote><p>אַךְ זֹאת לְכָמֹנִי בְּלִי יֵאוֹת.</p></blockquote>
<blockquote><p>עֵת נִכְסַפְתִּי / לִרְאוֹתֶךָ –</p></blockquote>
<blockquote><p>הִסְתּוֹפַפְתִּי / אֶל בֵּיתֶךָ.</p></blockquote>
<br/>
<blockquote><p>וָאֶסְבֹּלָה סֵבֶל עֲוֹן עַמִּי,</p></blockquote>
<blockquote><p>וְאַט אֱלֵי עֹל סֻבֳּלוֹ שִׁכְמִי,</p></blockquote>
<blockquote><p>לֹא אֶפְרֹשָׂה כַפִּי פְּנֵי אֶל זָר,</p></blockquote>
<blockquote><p>כִּי לֹא בְזוּלָתְךָ אֲנִי נֶעְזָר.</p></blockquote>
<blockquote><p>אִם נֶעְבַּדְתִּי / עַל־דָּתֶךָ –</p></blockquote>
<blockquote><p>לֹא עָבַדְתִּי / זוּלָתֶךָ.</p></blockquote>
<br/>
<blockquote><p>דּוֹדִי, אֲשֶׁר יָדִי בְּךָ תָרוֹם,</p></blockquote>
<blockquote><p>דַּלּוּ לְךָ עֵינַי אֱלֵי מָרוֹם,</p></blockquote>
<blockquote><p>עֻזִּי וּמְעֻזִי בְּךָ סֶלָה,</p></blockquote>
<blockquote><p>הֵן תִּקְטְלֵנִי – לְךָ אֲיַחֵלָה!</p></blockquote>
<blockquote><p>הֵן קִבַּלְתִּי / עֶבְרָתֶךָ –</p></blockquote>
<blockquote><p>כֵּן יִחַלְתִּי / חֶמְלָתֶךָ!</p></blockquote>
<br/>
<blockquote><p>הִתְאַמְּצָה נַפְשִׁי בְּעַבְדוּתְךָ,</p></blockquote>
<blockquote><p>וַתַּחֲזֵק בִכְנַף יְדִידוּתְךָ,</p></blockquote>
<blockquote><p>כֹּל עָבְרוּ עָלַי חֲרוֹנֶיךָ –</p></blockquote>
<blockquote><p>לֹא שָכְחָה נַפְשִׁי רְצוֹנֶךָ,</p></blockquote>
<blockquote><p>הִתְחַזַּקְתִּי / בִּבְרִיתֶךָ,</p></blockquote>
<blockquote><p>מָה‑ אָהַבְתִּי / תּוֹרָתֶךָ!</p></blockquote>
<br/>
