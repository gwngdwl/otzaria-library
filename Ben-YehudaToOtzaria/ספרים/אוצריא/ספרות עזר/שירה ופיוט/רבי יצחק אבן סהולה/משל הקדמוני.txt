<h1>משל הקדמוני</h1>
רבי יצחק אבן סהולה
שים לב! ספר זה אינו מושלם עדיין.
<h2>שער ראשון</h2>
<h3>אָמְרוּ הַקַּדְמוֹנִים (מתוך השער הראשון)</h3>
<blockquote><p>אָמְרוּ הַקַּדְמוֹנִים / וּפוֹתְרֵי הַדִּמְיוֹנִים /</p></blockquote>
<blockquote><p>כִּי הָיָה בְאֶרֶץ צֹעַן אִישׁ מַשְׂכִּיל בְּעִנְיָנָיו, / אִישׁ חָסִיד</p></blockquote>
<blockquote><p>אִישׁ עָנָו. / וַהֲוָת שַׁעְתָּא דְחִיקָא לֵהּ / וּבָנָיו בְּנֵי דְּלָא</p></blockquote>
<blockquote><p>מַעֲלֵי, / לֹא הָיָה מַשִּׂיג דֵּי מַחְסוֹרוֹ / בְּרֹב חָכְמָתוֹ</p></blockquote>
<blockquote><p>וְיָשְׁרוֹ, / וְהָיָה חוֹקֵר כָּל תַּעֲלוּמָה / בְּרָעָב וּבְצָמָא, /</p></blockquote>
<blockquote><p>וְכָל הַיּוֹם צוֹעֵק וּמֵהִים / נָגוּעַ וּמֻכֵּה אֱלֹהִים, / וּבָנָיו</p></blockquote>
<blockquote><p>חוֹזְרִים עַל הַפְּתָחִים, / נֶאֱנָקִים וְנֶאֱנָחִים. / וְהָיוּ</p></blockquote>
<blockquote><p>שְׁכֵנָיו שְמֵחִים לְאֵידוֹ / וּמִתְעַנְּגִים בְּפִידוֹ, / כִּי אָמְרוּ:</p></blockquote>
<blockquote><p>מַה לּוֹ וְלַחָכְמָה / וְלַחְקֹר תַּעֲלוּמָה? / וּכְבָר קָדַם הַנָּבִיא</p></blockquote>
<blockquote><p>בִּנְבוּאָתוֹ / ״אַל יִתְהַלֵּל חָכָם בְּחָכְמָתוֹ!״</p></blockquote>
<blockquote><p>וַיְהִי כַאֲשֶׁר גָּדַל כְּאֵבוֹ וְצַעֲרוֹ / וְהַקַּיִץ</p></blockquote>
<blockquote><p>חִלֵּל נִזְרוֹ / וַיִּתְחַזֵּק בַּשָּׁנָה הַהִיא חֹם גָּדוֹל וְהַיֹּבֶשׁ</p></blockquote>
<blockquote><p>רַד, / כִּי חָדַל הַמָּטָר וְהַבָּרָד – / וַיֶּאֱרַע לוֹ חֳלִי הַיֹּבֶשׁ</p></blockquote>
<blockquote><p>בְּמֹחוֹ, / סָר כֹּחוֹ: / וְהוּא חֳלִי הַמַּחֲשָׁב וְהַזִּכָּרוֹן /</p></blockquote>
<blockquote><p>וְאֵין לוֹ עַל הַבְּהֵמָה מַעֲלָה וְיִתְרוֹן. / וַיָּחֶל הַיָּגֵעַ /</p></blockquote>
<blockquote><p>הוֹלֵךְ וּמִשְׁתַּגֵּעַ, / וַתִסָּתֵר מִלִּבּוֹ הַחָכְמָה וְהַתְּשׁוּקָה</p></blockquote>
<blockquote><p>/ וְנַפְשׁוֹ שׁוֹקֵקָה, / וַיַּחֲזֵק בִּבְגָדָיו הַצְּנוּעִים /</p></blockquote>
<blockquote><p>וַיִּקְרָעֵם לִשְׁנֵים עָשָׂר קְרָעִים. / וַיִּתְלַקְּטוּ אֵלָיו</p></blockquote>
<blockquote><p>אֲנָשִׁים רֵיקִים וּפוֹחֲזִים, / שׁוֹדְדֵי עֶרֶב וּבוֹזֲזִים, /</p></blockquote>
<blockquote><p>וַיסוֹבֲבוּהוּ הַנְּעָרִים בְּרִיבוֹ / בְּשָׁמְעָם צַעֲקוֹ וְנִיבוֹ, /</p></blockquote>
<blockquote><p>וְתַעֲלוּלִים מָשְׁלוּ בוֹ. / וַיָּבוֹא עַד לִפְנֵי שַׁעַר הַמֶּלֶךְ /</p></blockquote>
<blockquote><p>מְצַחֵק וְתוֹמֵךְ בַּפֶּלֶךְ – / וְהַמֶּלֶךְ יוֹשֵׁב בַּעֲלִיַּת</p></blockquote>
<blockquote><p>הַמְּקֵרָה אֲשֶׁר לוֹ / וְהַשֵּׁגַל יוֹשֶׁבֶת אֶצְלוֹ. / וַיִּתְאַו</p></blockquote>
<blockquote><p>לִשְׁמֹעַ רִבּוּי אֲמָרָיו אֲשֶׁר לָהַג, / כִּי בְשִׁגָּעוֹן יִנְהַג, /</p></blockquote>
<blockquote><p>וַיְצַו לַהֲבִיאוֹ בִּרְגִילוּת / וְלֶאֱחֹז בְּסִכְלוּת, /</p></blockquote>
<blockquote><p>וַיִּשְׁתַעֲשַׁע הַמֶּלֶךְ בְּמַהֲלָלָיו / וְהַשֵּׁגַל בְּמַעֲלָלָיו. /</p></blockquote>
<blockquote><p>וַיִּתֶּן לוֹ הַמֶּלֶךְ כֻּתָּנְתּוֹ / וְלֶחֶם אָמַר לוֹ לְהַבְרוֹתוֹ, /</p></blockquote>
<blockquote><p>וַיַּפְקִידֵהוּ עִם שָׂרָיו הַנִּכְבָּדִים / וַיַּלְבִּישֵׁהוּ בְגָדִים,</p></blockquote>
<blockquote><p>/ וַיְהִי לִפְנֵיהֶם כָּל הַזְּמַנִּים / שׁוֹתֶה מַמְתַּקִּים וְאוֹכֵל</p></blockquote>
<blockquote><p>מַעֲדָנִים. / וּכְמוֹ כֵן צִוָּה לָתֵת / לְאִשְׁתּוֹ וּלְבָנָיו</p></blockquote>
<blockquote><p>מַשְׂאֵת. / וַיְהִי יוֹשֵׁב בְּמוֹשַׁב הַשָּׂרִים / עִם בְּנֵי הַמֶּלֶךְ</p></blockquote>
<blockquote><p>וְהַנְּעָרִים, / כִּי הָיָה מְשַׁעַשְׁעָם בְּדִבְרֵי הִתּוּל וּשְׂחוֹק /</p></blockquote>
<blockquote><p>כְּמִשְׁפַּט כָּל כְּסִיל וּכְחֹק. / וְכָל יוֹשְׁבֵי הָעִיר, / יַחַד</p></blockquote>
<blockquote><p>יָשִׁישׁ וְצָעִיר, / יְשִׂימוּן עַיִן עָלָיו לְשָׁמְרֵהוּ, / כִּי</p></blockquote>
<blockquote><p>מִצְוַת הַמֶּלֶךְ הוּא. / וַיֹּאמְרוּ כָּל יוֹדְעָיו כַּאֲשֶׁר רָאוּ</p></blockquote>
<blockquote><p>מַעֲלָתוֹ / בַּחֲצַר הַמֶּלֶךְ עִם כָּל בֵּיתוֹ / ״לְטוֹבָתוֹ /</p></blockquote>
<blockquote><p>נִשְׁבְּרָה רֶגֶל פָּרָתוֹ!״</p> <br/></blockquote>
<blockquote><p>וַיְהִי מִקֵּץ שְׁנָתַיִם יָמִים – וַיֵּעָתֵר אֱלֹהִים</p></blockquote>
<blockquote><p>לָאֲדָמָה / תַּחַת אֲשֶׁר הָיְתָה שְׁמָמָה / וְהִשְׁגִּיחַ עַל כָּל</p></blockquote>
<blockquote><p>יוֹשְׁבֶיהָ / מִמְּכוֹן עֲרָבוֹת, / וַיָּנֶף גִּשְׁמֵי נְדָבוֹת /</p></blockquote>
<blockquote><p>לְטוֹבָה וּלְשֹׂבַע וְלִבְרָכָה / שְׁמוּרָה בַכֹּל וַעֲרוּכָה. /</p></blockquote>
<blockquote><p>וַיָחֵלוּ הַנִּצָּנִים לְהִתְנָאוֹת / לַעֲלוֹת וְלֵרָאוֹת / וְלִהְיוֹת</p></blockquote>
<blockquote><p>כָּל עֵץ פּוֹרֵחַ, / הַגְּפָנִים וְהַסְּמָדַר נָתְנוּ רֵיחַ. / וַיָּקָם</p></blockquote>
<blockquote><p>הַיֹּבֶשׁ וַיִּבְרַח / וַיָּשִׂישׂ כְּגִבּוֹר לָרוּץ אֹרַח, / וַיָּבוֹא</p></blockquote>
<blockquote><p>חֹדֶשׁ הָאָבִיב בְּשִׂמְחָה / וְנָסוּ יָגוֹן וַאֲנָחָה. / וַיִּכָּנֵס</p></blockquote>
<blockquote><p>הַשֶּׁמֶשׁ בְּמַזַּל טָלֶה / וְקָרְאוּ אַחֲרָיו ״מָלֵא!״ / בִּהְיוֹתוֹ</p></blockquote>
<blockquote><p>מַעֲלֶה דְשָׁאִים / בִּזְרוֹעוֹ יְקַבֵּץ טְלָאִים. / וְהַיּוֹם</p></blockquote>
<blockquote><p>וְהַלַּיְלָה בְּאַהֲבָה נֶחְבָּקוּ, / אִישׁ בְּאָחִיהוּ יְדֻבָּקוּ /</p></blockquote>
<blockquote><p>וְנִשְׁתַּוּוּ בְּמַעֲלָתָם וְגָבְהָם / בְּמַרְאֵיהֶם וְגֹבַהּ לָהֶם. /</p></blockquote>
<blockquote><p>וַיִּרְבֶּה הַזַּךְ וְהַנָּקִי מִן הַדָּם / וַיָּשָׁב עֵין כָּל בָּחוּר</p></blockquote>
<blockquote><p>מְאָדָּם, / וְנִרְאָה כָּל לָבָן אֲדַמְדַּם, / וְהַמָּרָה הַשְּׁחֹרָה</p></blockquote>
<blockquote><p>לֹא תֵחָשֵׁב / וְנִקָּתָה לָאָרֶץ תֵּשֵׁב. / וְהַטֶּבַע הֶחָכָם מֵסִיר</p></blockquote>
<blockquote><p>הַדְּאָגוֹת / וּמְשַׁוֶּה הַזּוּגוֹת / וְהָעֲמֵלִים מִתְחַזְּקִים</p></blockquote>
<blockquote><p>לִשְׂמֹחַ וַעֲמָלָם לִשְׁכֹּחַ. / וַיֶּחְסַר בָּעֵת הַהִיא הַכֶּסֶל</p></blockquote>
<blockquote><p>וַיִּגָּרַע / וְאָסַף הַמְּצֹרָע – / וַיָּשָׁב הַכְּסִיל לְדַעְתּוֹ</p></blockquote>
<blockquote><p>וּלְעִנְיָנוֹ / בִּפְנוֹת בֹּקֶר לְאֵיתָנוֹ.</p> <p>וַיְהִי לְעֵת עֶרֶב / בְּהִתְעַצֵּב כָּל לֵב וְקֶרֶב. /</p></blockquote>
<blockquote><p>וַיִּדְרֹשׁ הַמֶּלֶךְ בְּעַד סְכָלוֹ / וַיַּרְא כִּי לֹא יָכוֹל לוֹ. /</p></blockquote>
<blockquote><p>וַיִּשְׁאָלוּהוּ הַשָּׂרִים וְהַשָּׂרוֹת / שִׂיחָה עֲמֻקָּה פִּי זָרוֹת,</p></blockquote>
<blockquote><p>/ וַיִּתְהוֹלְלוּ עִמּוֹ – וְלֹא עָנָה / וְאֶל שָׂטֵי כָזָב לֹא פָנָה./</p></blockquote>
<blockquote><p>וַיְהִי כַאֲשֶׁר הֶאֱלִיצוּהוּ / וּבְדִבְרֵי פַחֲזוּתָם הֱאִיצוּהוּ, /</p></blockquote>
<blockquote><p>וַיָּרֶם קוֹלוֹ וַיִּקְרָא בְשַׁוְעוֹ / ״אַךְ אֱוִילִים שָׂרֵי צֹעַן,</p></blockquote>
<blockquote><p>חַכְמֵי יוֹעֲצֵי פַרְעֹה!״ / וַיִּתְנַכֵּר אֲלֵיהֶם בְּמִלּוּלוֹ /</p></blockquote>
<blockquote><p>וּבֵאֵר שִׂיחוֹ הֵרִים קוֹלוֹ / וַיִּפְתַּח וַיִּשָּׂא מְשָׁלוֹ:</p> <br/></blockquote>
<blockquote><p>          תְרִיבוּנִי עֲלֵי חֶמְדַּת אֲמָרַי – / תְּבַקְשׁוּ חֵטְא וְסִכְלוּת מִדְּבָרַי:</p> <p>          הֲלֹא סָכָל וְאִישׁ הוֹלֵל – בְּצָרַי, / בְּנֵי הֶבֶל, אֲשֶׁר שׂוֹנְאַי וְעָרַי!</p> <p>          לְמָדוּנִי מְלִיצָה מִנְּעוּרַי, / וְהַשֵּׂכֶל וְהַבִּין – מִבְּחוּרַי.</p> <br/></blockquote>
<blockquote><p>וַיִּתְעַצְּבוּ הָאֲנָשִׁים עַל מַאֲמָרוֹ / כִּי חִלֵּל</p></blockquote>
<blockquote><p>דְּבָרוֹ / וַיֹּאמְרוּ ״לֹא בָא זֶה אֶלָּא לְרַמּוֹתֵנוּ / לִרְאוֹת</p></blockquote>
<blockquote><p>עֶרְוָתֵנוּ, / וְכָל סִכְלוּתוֹ הָיְתָה בְעָרְמָה / עַל פִּיו הָיְתָה</p></blockquote>
<blockquote><p>שׂוּמָה, / כַּאֲשֶׁר הֶעֱרִים הַמְשׁוֹרֵר נְאוּמוֹ / וַיְשַׁנֶּה אֶת</p></blockquote>
<blockquote><p>טַעְמוֹ / וַיּוֹרֶד רִירוֹ עַל זְקָנוֹ / וַיִּתְהוֹלֵל בִּרְצוֹנוֹ. /</p></blockquote>
<blockquote><p>וְהִנֵּה הוּא חַיָּב מַלְקוֹת אַרְבָּעִים / כְּמִשְׁפַּט הַמַּטְעִים״. /</p></blockquote>
<blockquote><p>וַיֹּאמְרוּ אֶל הַשּׁוֹטֵר לְהַלְקוֹתוֹ / וְכַמִּסְפָּר הַזֶּה</p></blockquote>
<blockquote><p>לְהַכּוֹתוֹ.</p> <br/></blockquote>
<blockquote><p>וַיְצַו הַמֶּלֶךְ לְהָסִיר מֵעָלָיו מַחֲלָצוֹת</p></blockquote>
<blockquote><p>הַטּוֹבוֹת / וַיַּלְבִּישֵׁהוּ בְלוֹיֵי הַסְּחָבוֹת / וּכְנֵצֶר נִתְעָב</p></blockquote>
<blockquote><p>מִבֵּיתוֹ הֻשְׁלַךְ / וַיְגָרְשֵׁהוּ וַיֵּלַךְ. / וַיֵּלֶךְ הָאִישׁ אֶל</p></blockquote>
<blockquote><p>בֵּיתוֹ סַר וְזָעֵף / יָגֵעַ וְעָיֵף, / וַיַּגֵּד לְאִשְׁתּוֹ וּלְבָנָיו</p></blockquote>
<blockquote><p>מַחֲלָתוֹ וְשִׁבְיוֹ / בַּחֲלוֹתוֹ וַיְחִי מֵחָלְיוֹ / וְאֶת כָּל אֲשֶׁר</p></blockquote>
<blockquote><p>אָמְרוּ הַשָּׂרִים לְהַכּוֹתוֹ / לְמַעַן עַנּוֹתוֹ. / וַיֹּאמַר ״עַתָּה</p></blockquote>
<blockquote><p>יָדַעְתִּי / וּסְבָרָתִי סָבַרְתִּי / כִּי הֶעְמִיק הֶחָכָם</p></blockquote>
<blockquote><p>בְּמִשְׁפָּטוֹ אֲשֶׁר שָׁפַט: / עָקָר מֵחָכְמָה מִכָּבוֹד סִכְלוּת</p></blockquote>
<blockquote><p>מְעַט!״</p> <br/></blockquote>
<h2>שער שני</h2>
<h3>אָמְרוּ כִּי הָיָה בְאַשְׁדּוֹד (מתוך השער השני - החסיד ואשתו הנואפת)</h3>
<blockquote><p>אָמְרוּ כִּי הָיָה בְאַשְׁדּוֹד אִישׁ תָּמִים</p></blockquote>
<blockquote><p>בְּמִדּוֹתָיו / וְצַדִּיק הָיָה בְּדוֹרוֹתָיו, / אִישׁ יִשְׁמָעאֵל /</p></blockquote>
<blockquote><p>נֶאֱמָן עִם אֵל / וְעִם קְדוֹשִׁים / זָקֵן בָּא בַאֲנָשִׁים. / וַתְּהִי</p></blockquote>
<blockquote><p>לוֹ אִשָּׁה מְיֻחֶסֶת וְתַמָּה, / יָפָה כַלְּבָנָה בָּרָה כַּחַמָּה, /</p></blockquote>
<blockquote><p>וּמֵרֹב צִדְקָתוֹ הָיָה נֶחְפָּר וְנִכְלָם / וְלֹא נִסְתַּכֵּל בְּהוֹד</p></blockquote>
<blockquote><p>יָפְיָהּ לְעוֹלָם / וּבְחֶזְקַת אֵשֶׁת חַיִל מַחֲזִיק אוֹתָהּ, / וְהוּא</p></blockquote>
<blockquote><p>לֹא רָאָה אֶת עֶרְוָתָהּ / וְלֹא הִכִּיר בִּפְעָלֶיהָ, / כִּי לֹא</p></blockquote>
<blockquote><p>הִבִּיט אֵלֶיהָ. / וַתַּחְשֹׁק הָאִשָּׁה בָּחוּר אֶחָד מִבְּנֵי הָעִיר /</p></blockquote>
<blockquote><p>אַדְמוֹנִי יְפֵה תֹאַר רַךְ בַּשָּׁנִים וְצָעִיר, / וַתִּבְעַר</p></blockquote>
<blockquote><p>בְּלִבָּהּ אֵשׁ תְּשׁוּקָתוֹ – / נִפְלְאַתָה אַהֲבָתוֹ / מַיִם רַבִּים</p></blockquote>
<blockquote><p>לֹא יָכְלוּ לְכַבּוֹת אַהֲבָתוֹ.</p> <p>וַיְהִי בִימֵי קְצִיר חִטִּים / וְרַעְיוֹנֵי הָאִשָּׁה</p></blockquote>
<blockquote><p>אֶל תַּאֲוָתָהּ מָטִים / וּמִדֶּרֶךְ הַצְּנִיעוּת נָדִים וּמִתְמוֹטֲטִים</p></blockquote>
<blockquote><p>/ כְּמַעֲשֵׂה שִׁטִּים. / וַיִּגַּע עֶרֶב חַגָּם בְּבִטְחָה – / הֶחָלִיל</p></blockquote>
<blockquote><p>וְהַשִּׂמְחָה – / וַיִּהְיוּ שְׂמֵחִים וְעַלִּיזִים, / וַיִּפְקֹד</p></blockquote>
<blockquote><p>הָאִישׁ אֶת אִשְׁתּוֹ בִּגְדִי הָעִזִּים. / וַיְהִי כַּעֲלוֹת הַמִּנְחָה</p></blockquote>
<blockquote><p>/ לְעֵת מְצֹא נֹפֶשׁ וַהֲנָחָה / וַיֵּלֶךְ הָאִישׁ בֵּית הַתְּפִלָּה /</p></blockquote>
<blockquote><p>בְּמַעֲטֵה עֹז וּתְהִלָּה / בְּגִילָה וְצָהֳלָה.</p> <p>וַיֹּאמֶר לְאִשְׁתּוֹ ״הָכִינִי הַשֻּׁלְחָן</p></blockquote>
<blockquote><p>וְהַמְּנוֹרָה / וְהִנְנִי הוֹלֵךְ לְהִתְפַּלֵּל בַּכְּנֶסֶת</p></blockquote>
<blockquote><p>הַהֲדוּרָה״.</p> <p>וַתַּעֲמֹד אִשְׁתּוֹ לְהָבִין בַּבָּנִים חֲסַר לֵב</p></blockquote>
<blockquote><p>וּבַעַר / נֹכַח פֶּתַח הַשַּׁעַר – / וְהִנֵּה הַבָּחוּר בְּרִנָּה /</p></blockquote>
<blockquote><p>עוֹבֵר בַּשּׁוּק אֵצֶל פִּנָּה. / וַיִּצְעַד וַיָּבוֹא לִקְרָאתָהּ /</p></blockquote>
<blockquote><p>וַיִּקְרַב אֶל פֶּתַח בֵּיתָהּ, / וַתִּסְגֹּר הָאִשָּׁה בַּעֲדוֹ /</p></blockquote>
<blockquote><p>וַתִּתְפְּשֵׂהוּ בְּבִגְדוֹ. / וַתֵּרֶא אֶת הוֹד יָפְיוֹ וְגָדְלוֹ, /</p></blockquote>
<blockquote><p>הֵעֵזָה פָנֶיהָ וַתֹּאמַר לוֹ / ״זִבְחֵי שְׁלָמִים עָלַי מִנְּעוּרַי /</p></blockquote>
<blockquote><p>הַיּוֹם שִׁלַּמְתִּי נְדָרַי. / וְהָיִיתִי עֲצוּבָה עַל נְדָרַי /</p></blockquote>
<blockquote><p>וְלִבִּי הַמָּוֶת בּוֹחֵר / לְמֵיקַם עֲלֵהּ מִשּׁוּם ׳בַּל תְּאַחֵר׳. /</p></blockquote>
<blockquote><p>וּכְבָר הִזְהִירוּנִי חַכְמֵי הַדּוֹרוֹת / לְהָבִין וּלְהוֹרוֹת /</p></blockquote>
<blockquote><p>וְאָמְרוּ: אֶחָד הַנּוֹדֵר וְאֶחָד הַמַּעֲרִיךְ / וּמִלְּקַיֵּם</p></blockquote>
<blockquote><p>מַאֲמָרוֹ הֶאֱרִיךְ / עוֹבֵר בְּ׳בַל תְּאַחֵר׳ אִם אֵינוֹ מַשְׁלִים /</p></blockquote>
<blockquote><p>כֵּיוָן שֶׁעָבְרוּ עָלָיו שָׁלֹשׁ רְגָלִים. / וְאָמְרוּ: חֹמֶר</p></blockquote>
<blockquote><p>בִּנְדָרִים מִשְּׁבוּעוֹת / וְכָל אֵלּוּ הֲלָכוֹת יְדוּעוֹת /</p></blockquote>
<blockquote><p>שֶׁהַנְּדָרִים עַל גַּב שְׁבוּעוֹת עוֹלִים / וּלְבַטֵּל הַמִּצְוָה</p></blockquote>
<blockquote><p>חָלִים. / וְאָמְרוּ: בַּעֲווֹן נְדָרִים אִשְׁתּוֹ שֶׁל אָדָם מֵתָה /</p></blockquote>
<blockquote><p>כִּי הִיא מִצְוָה צְרוּפָה וְעַל לוּחַ לֵב חֲרוּתָה. / וְעַתָּה אַחֲרֵי</p></blockquote>
<blockquote><p>אֲשֶׁר תּוֹחַלְתִּי מִמְּךָ לֹא אָבָדָה, / בּוֹא אִתִּי הַבַּיְתָה</p></blockquote>
<blockquote><p>וְנִסְעָדָה, / וְתִשְׂבַּע מֵהוֹדִי וּמִזִּיו פָּנַי / וַאֲשַׁלֵּם אֶת</p></blockquote>
<blockquote><p>נְדָרַי אֲשֶׁר נָדַרְתִּי לַייָ״. / וַיֹּאמֶר הַבָּחוּר ״הָיָה לָךְ</p></blockquote>
<blockquote><p>לָדוּן אוֹתִי מִקַּל וָחֹמֶר, כִּי אֲנִי תָאֵב מְאֹד לַחֲבוּרָתֵךְ /</p></blockquote>
<blockquote><p>וּפִי פָעַרְתִּי לְמַלְקוֹשׁ יְדִידוּתֵךְ – / אֲבָל לֹא רָאִיתִי שְׁעַת</p></blockquote>
<blockquote><p>הַכֹּשֶׁר / בֶּאֱמֶת וּבְיֹשֶׁר, / וַאֲנִי מְפַחֵד מִבַּעַל הַבַּיִת /</p></blockquote>
<blockquote><p>אֲשֶׁר קִנְאָתוֹ שָׁמִיר וָשַׁיִת״. / וַתֹּאמֶר הָאִשָּׁה ״הֲבִינוֹתִי</p></blockquote>
<blockquote><p>הוֹד מַחְשְׁבוֹתֶיךָ / וְנִלְאֵיתִי נְשֹׂא עֹל אַהֲבָתֶךָ / וְלִבִּי</p></blockquote>
<blockquote><p>בַּעֲנַק חִשְׁקְךָ נַפְשִׁי הֶעֱנִיק / יוֹתֵר מִמַּה שֶּׁהָעֵגֶל רוֹצֶה</p></blockquote>
<blockquote><p>לִינֹק – פָּרָה רוֹצָה לְהָנִיק״.</p> <br/></blockquote>
<blockquote><p>וַתּוֹסֶף עוֹד לְפַתּוֹתוֹ בִּדְבָרֶיהָ / וַתַּחֲלִיק</p></blockquote>
<blockquote><p>עָלָיו אֲמָרֶיהָ / וַתֹּאמַר ״דַּע כִּי בַעְלִי – הַשֵּׁם יִשְׁמְרֵהוּ /</p></blockquote>
<blockquote><p>וְיוֹצְרוֹ מִבֶּטֶן יַעְזְרֵהוּ! – / הוּא בַעַל תְּשׁוּבָה / וְנֶפֶשׁ</p></blockquote>
<blockquote><p>נִכְבֶּדֶת וַחֲשׁוּבָה / וְנָקִי בְכָל עִנְיָנָיו, / וְעָנְתָה בּוֹ</p></blockquote>
<blockquote><p>הַכָּרַת פָּנָיו / כִּי לֹא נִסְתַּכֵּל בְּהוֹד יָפְיִי / לְהַשְׁלִים</p></blockquote>
<blockquote><p>אֶת נִשְׁיִי. / וְהוּא הַגָּדוֹל מִבְּנֵי מְתִיבְתָּא / מַקְדִּימֵי</p></blockquote>
<blockquote><p>וּמַחֲשִׁיכֵי לְבֵי כְנִישְׁתָּא / וְשָׁעָה אַחֲרֵיהֶם מִתְאַחֵר /</p></blockquote>
<blockquote><p>וּבַבֹּקֶר לִפְנֵיהֶם שׁוֹחֵר / מִתְפַּלֵּל וּמִתְחַנֵּן יוֹתֵר</p></blockquote>
<blockquote><p>מֵחֲבֵרָיו / מְבָרֵךְ לְפָנָיו וּלְאַחֲרָיו / בַּקָּשׁוֹת רַבּוֹת</p></blockquote>
<blockquote><p>וְתַחֲנוּנִים / וּמַרְבֶּה רְנָנִים. / עַתָּה בּוֹא אִתִּי אֶל הַבִּירָה</p></blockquote>
<blockquote><p>/ סוּרָה, אֲדוֹנִי, אֵלַי אַל תִּירָא״. / וַיָּסַר אֵלֶיהָ אֶל הַחֶדֶר /</p></blockquote>
<blockquote><p>בַּעֲלִיַּת מִגְדָּל עֵדֶר / וַיֹּאכְלוּ וַיִּשְׁתּוּ וַיִּשְׂמָחוּ /</p></blockquote>
<blockquote><p>וַיַּעֲלוּ הָעוֹלָה וַיִּצְנָחוּ. / וַתֹּאמַר ״שְׂמַח בְּיַלְדוּתְךָ</p></blockquote>
<blockquote><p>וְאַל תִּתְעַצֵּב / הַשְׁכֵּם בַּבֹּקֶר וְהִתְיַצֵּב, / וְעַתָּה בּוֹא</p></blockquote>
<blockquote><p>אֵלַי, זוּלַת יָפְיִי אַל תַּחְמֹד / מְהֵרָה חוּשָׁה אַל תַּעֲמֹד, /</p></blockquote>
<blockquote><p>כִּי הָאִישׁ יַשְׁכִּים לִתְפִלָּתוֹ לְהַשְׁלִים תַּחֲנוּנָיו</p></blockquote>
<blockquote><p>כְּמִשְׁפָּטוֹ וְדָתוֹ / לַעֲשׂוֹת מַעֲשֵׂהוּ – זָר מַעֲשֵׂהוּ</p></blockquote>
<blockquote><p>נָכְרִיָּה עֲבוֹדָתוֹ! / וְעַתָּה, חֲשוּקִי, בְּמַאֲמָרִי בִסְתַּכֵּל</p></blockquote>
<blockquote><p>וַחֲשֹׁב / לֵךְ וָשׁוֹב!״</p> <p>וַיָסָב הַבָּחוּר אֶל בֵּית אָבִיו כְּחֻקּוֹ, /</p></blockquote>
<blockquote><p>וַתִּשָּׁאֵר הָאִשָּׁה לְכוּדָה בְחִשְׁקוֹ, / וַתְּמַהֵר וַתַּעַשׂ</p></blockquote>
<blockquote><p>מַטְעַמִּים / קִנְּמָן־בֶּשֶׂם וְסַמִּים / וַתְּכַבֵּד בֵּיתָהּ</p></blockquote>
<blockquote><p>בְּמִינֵי מַרְבַדִּים מְלֵאֵי עֵינַיִם / חֲטוּבוֹת אֵטוּן מִצְרַיִם, /</p></blockquote>
<blockquote><p>וַתַּעֲרֹךְ נֵרוֹתֶיהָ כְּפִי חֶפְצָהּ וּרְצוֹנָהּ, / טָבְחָה טִבְחָהּ</p></blockquote>
<blockquote><p>מָסְכָה יֵינָהּ / אַף עָרְכָה שֻׁלְחָנָהּ.</p> <p>וַיְהִי כְּכַלּוֹת הָאִישׁ לְהִתְפַּלֵּל כְּדָתוֹ /</p></blockquote>
<blockquote><p>וַיִּפֶן וַיָּבוֹא בֵית מְנוּחָתוֹ, / וַיִּמְצָא רְפִידָתוֹ עֲרוּכָה</p></blockquote>
<blockquote><p>וּשְׁמוּרָה, / מִטָּה שֻׁלְחָן כִּסֵּא מְנוֹרָה. / וַיֹּאמֶר לְאִשְׁתּוֹ</p></blockquote>
<blockquote><p>״בְּרוּכָה אַתְּ בְּפִי כָל עַם וְלָשׁוֹן! / הֵיטַבְתְּ חַסְדֵּךְ</p></blockquote>
<blockquote><p>הָאַחֲרוֹן מִן הָרִאשׁוֹן״. / וַיְמַהֵר וַיֵּשֶׁב בִּמְסִבּוֹ /</p></blockquote>
<blockquote><p>וַיֹּאכַל וַיֵּשְּׁתְּ וַיִּיטַב לִבּוֹ. / וַיְהִי כַּאֲשֶׁר כִּלָּה</p></blockquote>
<blockquote><p>לֶאֱכֹל פִּתּוֹ, / וַתֹּאמֶר לוֹ הָעֲנֻגָּה אִשְׁתּוֹ / ״הוֹאִיל</p></blockquote>
<blockquote><p>וְשָׂבַעְנוּ מִטּוּב הָאָרֶץ וְתִפְאַרְתָּהּ, / לֵימָא לָן מַר מִלֵּי</p></blockquote>
<blockquote><p>דַאֲגַדְתָּא!״ / וַיָּחֶל הָאִישׁ לְדַבֵּר / וּבְרִיחֵי אֱמוּנָתוֹ</p></blockquote>
<blockquote><p>לְשַׁבֵּר / וַיֹּאמֶר ״שִׁמְעִי, בִּתִּי, וּרְאִי וְהַטִּי אָזְנֵךְ /</p></blockquote>
<blockquote><p>לְמַאֲמַר יוֹצְרֵךְ וְקוֹנֵךְ, / וְאוֹדִיעֵךְ מוּסָרִים מְפֹרָשִׁים /</p></blockquote>
<blockquote><p>אֲשֶׁר בֵּאֲרוּ זְקֵנִים וִישִׁישִׁים / בְּעִנְיְנֵי הַנָּשִׁים. /</p></blockquote>
<blockquote><p>אָמְרוּ: צְרִיכָה הָאִשָּׁה לִהְיוֹת אֵשֶׁת חַיִל בְּמִדּוֹתֶיהָ, /</p></blockquote>
<blockquote><p>צְנוּעָה וּשְׁלֵמָה בְמַעֲלוֹתֶיהָ, / וּבַעֲבוֹדַת בַּעְלָהּ מְהִירָה</p></blockquote>
<blockquote><p>וּזְרִיזָה / וּבְאוֹר פָּנָיו מְאִירָה וּבְשִׂמְחָתוֹ עַלִּיזָה, /</p></blockquote>
<blockquote><p>וּלְהִתְבּוֹנֵן תָּמִיד בִּדְבָרוֹ וְאִמְרָתוֹ / כְּדֵי לְהָפִיק</p></blockquote>
<blockquote><p>רְצוֹנוֹ בַּעֲבוֹדָתוֹ / וְשֶׁלֹּא יִצְטָרֵךְ לִקְרֹאתָהּ פַּעֲמַיִם /</p></blockquote>
<blockquote><p>וְתִירָאֶנּוּ כְּמוֹרָא שָׁמַיִם / וְתִשְׁמֹר עֵת כַּעְסוֹ וְשִׂמְחָתוֹ</p></blockquote>
<blockquote><p>/ עֵת אַהֲבָתוֹ וְשִׂנְאָתוֹ / עֵת שִׁבְתּוֹ וְקוּמוֹ / עֵת קִיצָתוֹ</p></blockquote>
<blockquote><p>וְנוּמוֹ, / וְאַל תִּשְׂנָא אֲשֶׁר יֶאֱהַב לְמָנָה / וְאַל תֶּאֱהַב אֶת</p></blockquote>
<blockquote><p>אֲשֶׁר יִשְׂנָא, / וְאַל תֶּחֱשֹׂף סוֹדוֹתָיו / וְאַל תָּבוּז</p></blockquote>
<blockquote><p>מַעֲלוֹתָיו / תּוֹלְדוֹתָיו וּמִשְׁפְּחוֹתָיו. / וְאָמְרוּ: צְרִיכָה</p></blockquote>
<blockquote><p>לַעֲזֹב חֻקָּהּ וּמִנְהָגָהּ, / תַּכְשִׁיטֵי עֶדְיָהּ וְתַעֲנוּגָהּ, /</p></blockquote>
<blockquote><p>וְאַל תַּקְשֶׁה לִשְׁאֹל מִמֶּנּוּ / וּמַתָּנוֹ לְאַל אַל תְּשִׂימֶנּוּ,</p></blockquote>
<blockquote><p>/ וְכַאֲשֶׁר יִתֵּן לָהּ אַל תִּבְעַט / בֵּין רַב לִמְעַט, / וְתֶאֱהַב</p></blockquote>
<blockquote><p>מַאֲכָלוֹ וּמְזוֹנוֹ / כְּפִי חֶפְצוֹ וּרְצוֹנוֹ. / וּבְקִימָתָהּ</p></blockquote>
<blockquote><p>וְשִׁבְתָּהּ / תִּשְׁמֹר עִתּוֹ וְלֹא עִתָּהּ, / קַשֶּׁבֶת וְלֹא</p></blockquote>
<blockquote><p>דַבְּרָנִית / מְיֻשֶּׁבֶת וְלֹא יַצְאָנִית, / שֶׁעַל שֵׂכֶל וְדֵעָה /</p></blockquote>
<blockquote><p>חָצְבָה עַמּוּדֶיהָ שִׁבְעָה, / צוֹפִיָּה הֲלִיכוֹת בֵּיתָהּ וּמְעוֹנָהּ</p></blockquote>
<blockquote><p>/ וְתוֹרַת חֶסֶד עַל לְשׁוֹנָהּ. / וְאָמְרוּ: צְרִיכָה לְקַשֵּׁט</p></blockquote>
<blockquote><p>עַצְמָהּ דֶּרֶךְ צְנִיעוּת / כְדֵי שֶׁלֹּא יִמָּצֵא בָהּ דֹּפִי</p></blockquote>
<blockquote><p>וְטָעוּת, / וְאַל תִּהְיֶה עָלָיו הוֹמִיָּה וְסוֹרֶרֶת / וּבְיַחְסָהּ</p></blockquote>
<blockquote><p>מִשְׂתָּרֶרֶת, / וְאַל תַּרְאֶה לוֹ פָּנִים זוֹעֲפִים / וְרוּחַ קִנְאָה</p></blockquote>
<blockquote><p>שׁוֹאֲפִים, / וְיִגְדַּל בְּעֵינֶיהָ הוֹדוֹ וְגָדְלוֹ / בִּקְרָב אִישׁ</p></blockquote>
<blockquote><p>לְהִשְׁתַּחֲווֹת לוֹ. / וְכָל מוּם וְכָל אֵיבָה מִלִּבָּהּ תַּשְׁמִיד /</p></blockquote>
<blockquote><p>וְתִתְפַּלֵּל בַּעֲדוֹ תָמִיד, / וּכְבָבַת עֵינָהּ תִּנְצְרֵהוּ / כָּל</p></blockquote>
<blockquote><p>הַיּוֹם תְּבָרְכֵהוּ. / וְהָאִשָּׁה אֲשֶׁר אֵלֶּה אֳרָחֶיהָ – / עָצְמוּ</p></blockquote>
<blockquote><p>מִסַּפֵּר שְׁבָחֶיהָ, / וְעָלֶיהָ אָמְרוּ זִמְרָה, / רָחוֹק מִפְּנִינִים</p></blockquote>
<blockquote><p>מִכְרָהּ, / וְעַל שְׁלֵמוּת גָּדְלָהּ וּמַהֲלָלָהּ / בָּטַח בָּהּ לֵב</p></blockquote>
<blockquote><p>בַּעְלָהּ. / וְעַתָּה אֲכוֹנֵן לִבִּי וְרַעְיוֹנַי, / הַפַּעַם אוֹדֶה</p></blockquote>
<blockquote><p>אֶת יְיָ: / כִּי רֹב הַמִּדּוֹת אֲשֶׁר דִּבַּרְתִּי / וְהַמַּהֲלָל</p></blockquote>
<blockquote><p>אֲשְֶׁר זָכַרְתִּי – / לָךְ הֵם לְמִקְנָה / מִימֵי נְעוּרִים</p></blockquote>
<blockquote><p>וְעֶדְנָה״.</p> <p>וְכִשְׁמֹעַ הָאִשָּׁה אֶת דְּבָרָיו / וְנֹעַם אֲמָרָיו /</p></blockquote>
<blockquote><p>וַתָּקָם וַתַּחֲזֵק בְּבִגְדוֹ / וַתִּשְׁתַּחֲוֶה לְנֶגְדּוֹ / וַתֹּאמַר</p></blockquote>
<blockquote><p>״זֶהוּ מַאֲמַר חֲכָמַי וּנְבוֹנַי / זֶה הַשֻּׁלְחָן אֲשֶׁר לִפְנֵי יְיָ,</p></blockquote>
<blockquote><p>/ וְהִנֵּה אִם צָדַקְתָּ כִּי בְּמַהֲלָלְךָ תַּזְכִּירֵנִי – / אֶמְצָא</p></blockquote>
<blockquote><p>חֵן בְּעֵינֶיךָ, אֲדוֹנִי, לְהַכִּירֵנִי!״ / וַיְכַל הָאִישׁ עִנְיָנוֹ /</p></blockquote>
<blockquote><p>לְדַבֵּר עַל שֻׁלְחָנוֹ, / וַיָּקוּמוּ שְׁנֵיהֶם בְּחִפָּזוֹן / אַחַר</p></blockquote>
<blockquote><p>הַמַּאֲכָל וְהַמָּזוֹן / וַיָּסַר אֶל הָעֲלִיָּה וַיִּשְׁכְּבוּ שָׁם, /</p></blockquote>
<blockquote><p>וַיִּתְעַנְּגוּ כְּכָל תַּאֲוַת נַפְשָׁם. / וַיְהִי בְאַשְׁמֹרֶת</p></blockquote>
<blockquote><p>הַבֹּקֶר וַיֵּעוֹר הָאִישׁ מִשְׁנָתוֹ – / וַיָּפֹזוּ זְרוֹעֵי</p></blockquote>
<blockquote><p>תְנוּמָתוֹ. / וַיֹּאמַר ״אִשְׁתִּי, קוּמִי, אֲחוֹתִי, / הָכִינִי</p></blockquote>
<blockquote><p>שִׂמְלָתִי וּכְסוּתִי, / כִּי כְּבָר נִתְכְּנוּ רַעְיוֹנַי: / עֵת</p></blockquote>
<blockquote><p>לַעֲשׂוֹת לַייָ! / וְעַתָּה לְכַבֵּד בֵּיתֵךְ וְלַעֲרֹךְ מַאֲכָלֵךְ</p></blockquote>
<blockquote><p>עוּרִי / בְּחִטֵּי מִנִּית וּפַנַּג וּדְבַשׁ וָשֶׁמֶן וָצֹרִי, / וְאַחַר</p></blockquote>
<blockquote><p>תָּשִׂימִי לִבֵּךְ לַמְסִלָּה / לַעֲלוֹת אֶל בֵּית הַתְּפִלָּה / וְלָתֵת</p></blockquote>
<blockquote><p>הוֹדָאָה עַל טוּב חֶלְקֵנוּ / וְאוֹר נָגְהֵנוּ, / כִּי קָדוֹשׁ הַיּוֹם</p></blockquote>
<blockquote><p>לַאֲדוֹנֵינוּ״.</p> <p>וַתֹּאמֶר הָאִשָּׁה בְּנֶפֶשׁ נְכֵאָה / ״אַל תָּשֶׁת</p></blockquote>
<blockquote><p>בִּי עָווֹן וּשְׁגִיאָה / כִּי לֹא אֵצֵא מִבֵּיתִי, / פֶּן אַשְׁחִית</p></blockquote>
<blockquote><p>נַחֲלָתִי. / גְּאַל לְךָ אַתָּה אֶת גְּאֻלָּתִי / וְהִתְפַּלֵּל</p></blockquote>
<blockquote><p>בַּעֲדֵנוּ / לְהַגְדִּיל מַהֲלָלֵנוּ וְהוֹדֵנוּ, / כִּי בְנוֹת הָעִיר</p></blockquote>
<blockquote><p>הַזֹּאת / מְאֹד הֵן קָשׁוֹת וְעַזּוֹת: / כִּי כָל אַחַת מֵהֶן</p></blockquote>
<blockquote><p>לְתַאֲוָתָהּ שׁוֹחֶרֶת / וְעַל הוֹד בַּעְלָהּ אֲמָרֶיהָ מְמָרֶרֶת /</p></blockquote>
<blockquote><p>הוֹמִיָּה וְסוֹרֶרֶת, / מִתְעַסֶּקֶת בְּדִבְרֵי כְלִמָּה / פְּתַיּוּת</p></blockquote>
<blockquote><p>וּבַל יָדְעָה מָּה. / וְכָל אַחַת בְּדִבְרֵי חֲבֶרְתָּהּ הֶחֱזִיקָה /</p></blockquote>
<blockquote><p>נָכְרִיָּה – אֲמָרֶיהָ הֶחֱלִיקָה. / וְאֵין בֵּינֵיהֶן נְבוֹנָה / כִּי</p></blockquote>
<blockquote><p>אִם פּוֹתָה וְדִבְרֵי עֲגָבִים מוֹנָה, / שִׁית זוֹנָה. / וְלֹא</p></blockquote>
<blockquote><p>תִתְחַבַּרְנָה בְּבָתֵי הַתְּפִלּוֹת / לִשְׁמֹעַ תַּחֲנוּנִים</p></blockquote>
<blockquote><p>וּתְהִלּוֹת, / כִּי אִם לִפְנֵי הַבַּחוּרִים לְהִתְנָאוֹת / לַעֲלוֹת</p></blockquote>
<blockquote><p>וְלֵרָאוֹת / וְהֵם לִרְאוֹתָן יֶאֱרֹבוּ / וּלְמַלֹּאת פְּגִימָתָן</p></blockquote>
<blockquote><p>יַחֲשֹׁבוּ / וְיִרְמְזוּ בְעֵינַיִם וּבְגַבּוֹתָם / יִצְפְּנוּ</p></blockquote>
<blockquote><p>לְנַפְשׁוֹתָם. / וְכַאֲשֶׁר שָׁמַעְתִּי סוֹדֵיהֶם / וָאֶתְבּוֹנֵן</p></blockquote>
<blockquote><p>עָדֵיהֶם, / אָמַרְתִּי: חוֹבָה לְהִתְרַחֵק מִכָּל לֵב הֵזִיד וְשָׁגַג –</p></blockquote>
<blockquote><p>/ טוֹב לָשֶׁבֶת עַל פִּנַּת גָּג! / וְנִשְׁבַּעְתִּי, אֲדוֹנִי,</p></blockquote>
<blockquote><p>בְּחַיֵּי רֹאשְׁךָ / חַי יְיָ וְחֵי נַפְשֶׁךָ, / אִם אֵצֵא מִפֶּתַח</p></blockquote>
<blockquote><p>בֵּיתִי / וְעָנְתָה בִּי צִדְקָתִי! / כָּל שֶׁכֵּן שֶׁאֵין הָאִשָּׁה</p></blockquote>
<blockquote><p>נֶחְשֶׁבֶת / וּבְמִצְוַת הַיּוֹם אַינָהּ חַיֶּבֶת / וְכֵן פֵּרְשׁוּ</p></blockquote>
<blockquote><p>בַעֲלֵי הַמְּסוֹרוֹת: / כָּל מִצְוַת עֲשֵׂה שֶׁהַזְּמָן גְּרָמָהּ –</p></blockquote>
<blockquote><p>נָשִׁים פְּטוּרוֹת״. / וַיֹּאמֶר הָאִישׁ ״בָּרוּךְ יְיָ אֲשֶׁר נָתַן</p></blockquote>
<blockquote><p>כָּזֹאת בְּלִבֵּךְ / וְהֵבִיא רוּחַ חֵן וְתַחֲנוּנִים בְּקִרְבֵּךְ /</p></blockquote>
<blockquote><p>וְתָשִׂימִי לְאַל דִּבְרֵי כְסִילוֹת וְנִבְזוֹת: / לְזֹאת יִקָּרֵא</p></blockquote>
<blockquote><p>אִשָּׁה כִּי מֵאִישׁ לֻקֳחָה זֹּאת! / וְעַתָּה שְׁבִי בֵית</p></blockquote>
<blockquote><p>תִּפְאַרְתֵּךְ וְהוֹדֵךְ – / וַאֲנִי אֶתְפַּלֵּל בַּעֲדֵךְ״. / וַיֵּלֶךְ</p></blockquote>
<blockquote><p>הָאִישׁ לְדַרְכּוֹ / לְהִתְפַּלֵּל לִפְנֵי מַלְכּוֹ.</p> <p>וַיְהִי כַּאֲשֶׁר יָצָא מִבֵּית מְנוּחוֹ – / וְהַבָּחוּר</p></blockquote>
<blockquote><p>הִשְׁכִּים אֶל פִּתְחוֹ / וַיִּרְמֹז אֶל הָאִשָּׁה בְּמַטֵּהוּ, /</p></blockquote>
<blockquote><p>וַתִּפְתַּח וַתִּרְאֵהוּ / וַתֹּאמֶר לוֹ בְּקוֹל נָחוּץ / ״בּוֹא,</p></blockquote>
<blockquote><p>בְּרוּךְ יְיָ, לָמָּה תַעֲמֹד בַּחוּץ? / כִּי אֵין הָאִישׁ בְּבֵיתוֹ</p></blockquote>
<blockquote><p>וּבִזְבוּלוֹ / חָלַף וְהָלַךְ לוֹ״. / וַיָבוֹא וַיִּתְהַלֵּל</p></blockquote>
<blockquote><p>בַּעֲלוּמָיו / וַיִּגְרְשׁוּ מֵימָיו. / וַיֹּאכְלוּ וַיִּשְׁתּוּ כְּחֹק</p></blockquote>
<blockquote><p>הַחוֹשְׁקִים הַנֵּחָמִים / וַיִּצְנְחוּ לָאָרֶץ שֶׁבַע פְּעָמִים /</p></blockquote>
<blockquote><p>וַיִּשְׂבְּעוּ מַעֲדַנִּים לְפִי תַאֲוָתָם / הֵרֵעוּ מֵאֲבוֹתָם. /</p></blockquote>
<blockquote><p>וַיְהִי כַּאֲשֶׁר כִּלּוּ לֶאֱכֹל אֶת הַשֶּׁבֶר – / אֵשֶׁת מִדְיָנִים</p></blockquote>
<blockquote><p>וּבֵית חֶבֶר! – / וַיִּפֶן הַבָּחוּר אֶל דֶּרֶךְ גְּבוּלוֹ / וַיֵּלֶךְ</p></blockquote>
<blockquote><p>לוֹ. / וַתְּמַהֵר הָאִשָּׁה לְהָכִין מַאֲכָלָהּ וּמְזוֹנָהּ /</p></blockquote>
<blockquote><p>וַתִּתְעַטֵּף וַתֵּשֶׁב עַל מְכוֹנָהּ.</p> <p>וַיָּבוֹא אִישָׁהּ מִבֵּית מִדְרָשׁוֹ / הָדוּר</p></blockquote>
<blockquote><p>בִּלְבוּשׁוֹ, / וַתָּקָם לְפָנָיו בְּשִׁפְלוּת עֵינַיִם / וַתִּקֹּד בַּת</p></blockquote>
<blockquote><p>שֶׁבַע אַפַּיִם / וַתְּסִירֵהוּ אֶל מְקוֹמוֹ וַהֲדָרוֹ / וְאֶל מוֹשַׁב</p></blockquote>
<blockquote><p>יְקָרוֹ. / וַיָּבוֹאוּ אוֹהֲבָיו וְכָל בְּנֵי מִשְׁפַּחְתּוֹ /</p></blockquote>
<blockquote><p>וַיִּפְקְדוּ אוֹתוֹ וְאֶת אִשְׁתּוֹ / וַיֹּאמְרוּ ״הֲשָׁלוֹם לָכֶם מֵאֵל</p></blockquote>
<blockquote><p>נוֹרָא וְעֵילוֹם?״ / וַיֹּאמְרוּ ״שָׁלוֹם!״ / וַיְהִי אַחֲרֵי כֵן</p></blockquote>
<blockquote><p>וַיֵּשֶׁב הָאִישׁ אֶל מְזוֹנוֹ / וְהָאִשָּׁה כְּכֶתֶם אוֹפִיר לִימִינוֹ,</p></blockquote>
<blockquote><p>/ וַיִּשְׂמְחוּ בְחַגָּם וּבְמוֹעֲדָם / כִּי זֶה כָּל הָאָדָם – / חֹק</p></blockquote>
<blockquote><p>וּזְמָן נִתַּן לָהֶם שֶׁלֹּא יְשַׁנּוּ אֶת תַּפְקִידָם.</p> <br/></blockquote>
<blockquote><p>וַיִּרְבּוּ הַיָּמִים / בַּחֲטָאִים וַאֲשָׁמִים / וְלֹא</p></blockquote>
<blockquote><p>הֵבִין הָאִישׁ עִנְיְנֵי אִשְׁתּוֹ וּמַעְבָּדֶיהָ, / כִּי מְצוֹדִים</p></blockquote>
<blockquote><p>וַחֲרָמִים – לִבָּהּ, אֲסוּרִים – יָדֶיהָ. / וַיְהִי לִתְקוּפַת</p></blockquote>
<blockquote><p>הַשָּׁנָה / בְּהִתְעוֹרֵר לֵב כָּל זוֹנָה / וְכָל מַמְרִים / וּלְמַלֵּא</p></blockquote>
<blockquote><p>תַאֲוָתוֹ יַעְרִים – / וַיִּתְאַו הַבָּחוּר לְצַחֵק / וַיֵּלֶךְ מֵאָבִיו</p></blockquote>
<blockquote><p>וַיִּתְרַחֵק / וַיֵּצֵא בֵית הַשְּׂחוֹק / כְּמִשְׁפַּט כָּל זוֹנֶה</p></blockquote>
<blockquote><p>וּכְחֹק. / וַתִּפֹּל עָלָיו גָּדֵר הַדְּחוּיָה – / וַיְצַחֵק בְּקֻבְיָא</p></blockquote>
<blockquote><p>/ וַיְחַסֵּר הוֹנוֹ וּכְסוּתוֹ גָרַע / וְאָבַד הָעֹשֶׁר הַהוּא</p></blockquote>
<blockquote><p>בְּעִנְיַן רָע. / וַיִּתְחַבֵּא כָל הַיּוֹם הַהוּא לִבְרֹחַ /</p></blockquote>
<blockquote><p>וַיִּשְׁכַּח בֵּית אָבִיו שָׁכוֹחַ / וַיִּשְׂכֹּר לוֹ בְהֵמָה לִרְכַּב /</p></blockquote>
<blockquote><p>וְלָן וְשָׁכַב.</p> <p>וַיְהִי בַלַּיְלָה בְּהַגִּיעַ עֵת תְּפִלָּתָם וְעוֹנָה</p></blockquote>
<blockquote><p>/ רֹאשׁ הָאַשְׁמֹרֶת הַתִּיכוֹנָה / וַיֵּעוֹר הָאִישׁ מִשְּׁנָתוֹ /</p></blockquote>
<blockquote><p>וַיֵּלֶךְ לְהִתְוַדּוֹת עַל עֲווֹן אַשְׁמָתוֹ. / וַיַּשְׁכֵּם הַבָּחוּר</p></blockquote>
<blockquote><p>לָלֶכֶת לְדַרְכּוֹ / וְלָנוּס מֵעִירוֹ וּמִקְּהִלַּת פִּלְכּוֹ, /</p></blockquote>
<blockquote><p>וַיִּקְרָא אֶל הָאִשָּׁה אֶל הַפֶּתַח / לִפְתֹּחַ פֶּה בְּרֶצַח /</p></blockquote>
<blockquote><p>וַיֹּאמֶר לָהּ ״הִנְנִי הוֹלֵךְ מִבֵּיתִי וְאוּלַמִּי / אֶל מְלוֹן</p></blockquote>
<blockquote><p>אוֹרְחִים וְאֶעֶזְבָה אֶת עַמִּי״. / וַתֹּאמֶר לוֹ הָאִשָּׁה ״אַל</p></blockquote>
<blockquote><p>תִּמְנָעֵנִי, חֲשׁוּקִי, מֵעַצְמְךָ – / הָלוֹךְ אֵלֵךְ עִמְּךָ!״ /</p></blockquote>
<blockquote><p>וַתִּקַּח מִבֵּית בַּעְלָהּ כָּל אֲשֶׁר תֶּאֱהַב: / כֶּסֶף וְזָהָב, /</p></blockquote>
<blockquote><p>בְּגָדִים וּרְדִידִים, / נְזָמִים וּצְמִידִים. / וַיָּבוֹא הַבָּחוּר אֶל</p></blockquote>
<blockquote><p>הֵיכָלָהּ / וַיִּקַּח אֶת רְכוּשׁ בַּעְלָהּ, / עֲדָיִים וּשְׁבִיסִים, /</p></blockquote>
<blockquote><p>תִּפְאֶרֶת הָעֲכָסִים / כְּלִילָם וְאֶצְעָדָתָם וַעֲגִילָם / עֲשָׂרָה</p></blockquote>
<blockquote><p>זָהָב מִשְׁקָלָם. / וַיִּשְׂכְּרוּ לָהֶם בְּהֵמָה אַחֶרֶת / חֲזָקָה</p></blockquote>
<blockquote><p>וְנִבְחֶרֶת, / וַיִּקַּח כָּל אֶחָד מַשָּׂאוֹ וּמִבְחָרוֹ / וַיַּעֲמֹס</p></blockquote>
<blockquote><p>אִישׁ עַל חֲמוֹרוֹ, / וְלֹא הִנִּיחַ לְבַד הַכָּבֵד / וְכָל דָּבָר</p></blockquote>
<blockquote><p>הָאָבֵד / אֲשֶׁר לֹא הָיָה עוֹמֵס / וְכָל הַמְּלָאכָה נְמִבְזָה וְנָמֵס.</p></blockquote>
<blockquote><p>/ וַיָּקוּמוּ וַיֵּלְכוּ בַנֶּשֶׁף / מְזֵי רָעָב וּלְחוּמֵי רֶשֶׁף, /</p></blockquote>
<blockquote><p>וַיָּשִׂימוּ מְגַמַּת פְּנֵיהֶם אֶל אַדְמַת מִצְרַיִם לְחָגָּא / דֶּרֶךְ</p></blockquote>
<blockquote><p>אַשְׁדּוֹת הַפִּסְגָּה, / וַיָּסֹבּוּ אֶת כָּל עָרֵי הָאֱמוֹרִי</p></blockquote>
<blockquote><p>וּמִגְרָשָׁם / וַיָּבוֹאוּ עַד חָרְמָה וַיֵּשְׁבוּ שָׁם.</p> <br/></blockquote>
<blockquote><p>וַיְהִי בַבֹּקֶר עֵת שִׂמְחַת הַלְּבָבוֹת / בֹּקֶר לֹא</p></blockquote>
<blockquote><p>עָבוֹת – / וַיְכַל הָאִישׁ לְהִתְוַדּוֹת עַל מַעֲלוֹ – / דֹּם לַייָ</p></blockquote>
<blockquote><p>וְהִתְחוֹלֵל לוֹ – / וַיָּרָץ אֶל בֵּיתוֹ בִּמְרוּצָה / וּפֶתַח פָּתוּחַ</p></blockquote>
<blockquote><p>מָצָא. / וַיַּרְא – וְהִנֵּה בֵיתוֹ שָׁדוּד, אֵין בּוֹ מִחְיָה /</p></blockquote>
<blockquote><p>וְעֵירֹם וְעֶרְיָה. / וַיִּכָּנֵס אֶל גְּנָזָיו – וְהִנָּם נְסוֹגִים /</p></blockquote>
<blockquote><p>כַּמְּצוּלָה שֶׁאֵין בָּהּ דָּגִים. / וַיֹּאמַר: אֵין זֶה כִּי אִם</p></blockquote>
<blockquote><p>אִשְׁתִּי הַתְּמִימָה / הִשְׁאִילָה מֵיטַב חֲפָצֵינוּ לְכַלָּה יְתוֹמָה</p></blockquote>
<blockquote><p>/ כְּדֵי לִגְמֹל חֲסָדִים / בְּעֵת דּוֹדִים / (וְלֹא יָדַע הֶחָסִיד</p></blockquote>
<blockquote><p>הֶעָנָו / כִּי קָרַן עוֹר פָּנָיו!) / וְעַתָּה אֲיַחֵל עַד עֵת בּוֹאָהּ</p></blockquote>
<blockquote><p>/ וְאֵדְעָה מוֹצָאָהּ. / וַיְיַחֵל הָאִישׁ לִמְצֹא פִדְיוֹם / עַד נְכוֹן</p></blockquote>
<blockquote><p>הַיּוֹם / וַיַּרְא וְהִנָּהּ אֵחֲרָה לָבוֹא / וַיּתְעַצֵּב אֶל לִבּוֹ. /</p></blockquote>
<blockquote><p>וַיֹּאמַר: אֵלֵךְ לְהָבִין מִי הֶלְאַנִי / וְאֵדְעָה מֶה חָדֵל אָנִי. /</p></blockquote>
<blockquote><p>וַיָּקָם וַיִּדְרֹשׁ בַּעֲדָהּ בְּכָל מוֹשָׁב / נִכְבָּד לַכֹּל</p></blockquote>
<blockquote><p>וְנֶחְשָׁב – / וְלֹא מָצָא לִשְׁאֵלָתוֹ מַעֲנֶה / וְאֵין קוֹל וְאֵין</p></blockquote>
<blockquote><p>עוֹנֶה. / וַיָּבוֹא עַד לִפְנֵי שַׁעַר הָעִיר / שׁוֹאֵל לְכָל יָשִׁישׁ</p></blockquote>
<blockquote><p>וְצָעִיר, / וְהִנֵּה אִישׁ מִדֶּרֶךְ הַקָּדִים / בָּא וְהִקְדִּים /</p></blockquote>
<blockquote><p>וַיַּרְא אֶת הָאִישׁ וַיַּכִּירֵהוּ – / וַיִּקֶר מִקְרֵהוּ.</p> <p>וַיֹּאמַר ״הֲלֹא אַתָּה הָאִישׁ בַּעַל הַתְּשׁוּבָה /</p></blockquote>
<blockquote><p>הוֹלֵךְ בְּדֶרֶךְ חֲשׁוּבָה, / וְאִשְׁתְּךָ הָיְתָה בְּעֵינַי בְּחֶזְקַת</p></blockquote>
<blockquote><p>הַצְּנוּעוֹת – / וְהִנֵּה הִיא מִן הַזּוֹנוֹת הַגְּרוּעוֹת! / כִּי</p></blockquote>
<blockquote><p>הַיּוֹם פְּגַשְׁתִּיהָ עִם פְּלוֹנִי הַבָּחוּר / רָצִים בְּאֵין אִחוּר,</p></blockquote>
<blockquote><p>/ וַחֲמוֹרֵיהֶם נוֹשְׂאִים מַשָּׂא כִפְלַיִם / הוֹלְכִים לְהוֹרִיד</p></blockquote>
<blockquote><p>מִצְרַיִם. / וְאֵין אִישׁ יָכוֹל לְהַשִּׂיג אוֹתָם / לְתֹקֶף מַהֲלָכָם</p></blockquote>
<blockquote><p>וּמְרוּצָתָם״. / וַיְהִי כִשְׁמֹעַ הֶחָסִיד עִנְיָנָיו, / כִּסְּתָה</p></blockquote>
<blockquote><p>כְלִמָּה פָּנָיו / וַיִּתֵּן אֶת קוֹלוֹ בִּבְכִי וַיִּתְנַהֵג כְּשׁוֹלָל</p></blockquote>
<blockquote><p>/ וְנָאַק נַאֲקַת חָלָל. / וַיִּצְעַק וַיֹּאמַר ״הֵן בַּעֲווֹן צִדְקִי</p></blockquote>
<blockquote><p>נִטְמֵאתִי, / אוֹי לִי כִי נִדְמֵיתִי! / וְסִבַּת בָּשְׁתִּי</p></blockquote>
<blockquote><p>וּכְלִמָּתִי / וִיגוֹן לִבִּי וְחֶרְפָּתִי / הָיְתָה לִי דִמְעָתִי, /</p></blockquote>
<blockquote><p>וְעִיּוּן תְּפִלָּתִי וּתְשׁוּבָתִי – / הָיְתָה מַחֲלָתִי. / כִּי אִם</p></blockquote>
<blockquote><p>הִתְנַהַגְתִּי בְּגַאֲוָה וּבְזָדוֹן / וְהָיִיתִי בְּבֵיתִי אִישׁ מָדוֹן</p></blockquote>
<blockquote><p>– / הָיִיתִי מַכִּיר אֶת אִשְׁתִּי וְאוֹדוֹתֶיהָ / מַלְקָחֶיהָ</p></blockquote>
<blockquote><p>וּמַחְתּוֹתֶיהָ / וּזְדוֹנִי בְּכָל יוֹם יֵדָעֶנָּה / וְגַאֲוָתִי בְּכָל</p></blockquote>
<blockquote><p>עֵת תִּדְרְשֶׁנָּה / וַתַּשְׁחֵת עַגְבָתָהּ מִמֶּנָּה. / אֲבָל</p></blockquote>
<blockquote><p>פִּתַּתְנִי הַמְּשׁוּבָה וָאֶתְפַּתֶה / וָאֶמְצָא כוֹס חֲמַת אִשְׁתִּי</p></blockquote>
<blockquote><p>וָאֶשְׁתֶּה, / וַיִּכְלוּ בַהֶבֶל יָמַי / וַיָּבוֹא רָקָב בַּעֲצָמַי. /</p></blockquote>
<blockquote><p>וְעַתָּה, שָׁבִים, סוּרוּ מִדֶּרֶךְ תְּשׁוּבָה / וּבַחֲרוּ לָכֶם דֶּרֶךְ</p></blockquote>
<blockquote><p>עַזּוֹת וּמְשׁוּבָה, / פֶּן יִקְרֶה אֶתְכֶם כַּאֲשֶׁר קְרָאַנִי /</p></blockquote>
<blockquote><p>וְיִשְׁבֹּר הַדְרַתְכֶם הַיָּגוֹן אֲשֶׁר שְׁבָרַנִי / וְתִהְיוּ לִמְשֹׁל</p></blockquote>
<blockquote><p>עַמִּים וְקִינָה / לְחֶרְפָּה לְמָשָׁל וְלִשְׁנִינָה״.</p> <p>וַיָשָׁב הָאִישׁ לְבֵיתוֹ / כַּאֲשֶׁר כִּלָּה לְסַפֵּר</p></blockquote>
<blockquote><p>אַנְחָתוֹ / וַיִּשָּׁאֵר לְחֶרְפָּה וּלְבִזָּיוֹן / לְדַאֲבוֹן נֶפֶשׁ</p></blockquote>
<blockquote><p>וְכִלָּיוֹן. / וַיִּתֵּן תַּחַת שִׂמְחָתוֹ – אֲנָחָה / וּמְנוֹד רֹאשׁ –</p></blockquote>
<blockquote><p>תַּחַת הֲנָחָה / וְתַחַת מַעֲשֵׂה מִקְשֶׁה – קָרְחָה / וְתַחַת כָּבוֹד –</p></blockquote>
<blockquote><p>מוּם וָדֹפִי, / כִּי – תַחַת יֹפִי.</p> <br/></blockquote>
<h3>הָיָה אִישׁ אֶחָד בְּאֶרֶץ הַתֵּימָנִי (מתוך השער השני - השודד והנזיר)</h3>
<blockquote><p>מתוך השער השני [השודד והנזיר]</p> <br/></blockquote>
<blockquote><p>הָיָה אִישׁ אֶחָד בְּאֶרֶץ הַתֵּימָנִי / מִמִּשְׁפַּחַת</p></blockquote>
<blockquote><p>הַדָּנִי / וּשְׁמוֹ חֶבֶר הָאֲחוֹחִי / בֶּן-בִּלְדַּד הַשּׁוּחִי, /</p></blockquote>
<blockquote><p>וְלֹא לָמַד תּוֹרָה בְּקַטְנוּתוֹ / וְאֶל חֹק וּמִשְׁפָּט לֹא הִטָּה</p></blockquote>
<blockquote><p>אֱמוּנָתוֹ, / שָׁלַח יָדָיו בִּשְׁלוֹמָיו חִלֵּל בְּרִיתוֹ / וְהָיָה</p></blockquote>
<blockquote><p>מְזַלְזֵל אָבִיו וְאֶת בְּנֵי עַמּוֹ, / וּכְסִיל אָדָם בּוֹזֶה אִמּוֹ, /</p></blockquote>
<blockquote><p>וַיִּתְחַבֵּר אֶל בְּנֵי בְלִיַּעַל וַּיִּמְעֹל מַעַל. / וְכַאֲשֶׁר</p></blockquote>
<blockquote><p>רָאָה אָבִיו אָרְחוֹתָיו / מַהֲלָכָיו וּנְתִיבוֹתָיו, / הִדִּיחוֹ</p></blockquote>
<blockquote><p>מִבֵּיתוֹ לְרֹעַ מַעֲשָׂיו / וְנִדָּהוּ וְהוֹרִידוֹ מִנְּכָסָיו. /</p></blockquote>
<blockquote><p>וַיֵּצֵא הָאִישׁ מִבֵּית הָאִישׁ הַנֶּאֱמָן / אֶל מִדְבַּר צִן נֶגְבָּה</p></blockquote>
<blockquote><p>מִקְצֵה תֵימָן, / וַיְהִי לְכָל עוֹבֵר וָשָׁב מַשָּׂא וָטֹרַח / נָחָשׁ</p></blockquote>
<blockquote><p>עֲלֵי דֶרֶךְ שְׁפִיפוֹן עֲלֵי אֹרַח.</p> <p>וַיְהִי הַיּוֹם וְהִנֵּה נָזִיר אֶחָד עוֹבֵר / לְיִרְאֵי</p></blockquote>
<blockquote><p>אֱלֹהִים רֵעַ וְחָבֵר, / וַיִּתְפֹּשׂ בּוֹ הָאִישׁ לְבַהֲלוֹ / וְלִגְזֹל</p></blockquote>
<blockquote><p>הוֹנוֹ וְחֵילוֹ – / וְלֹא מָצָא בְיָדוֹ דָבָר, / כַּאֲשֶׁר חָשַׁב</p></blockquote>
<blockquote><p>וְסָבַר. / וַיְהִי כִרְאוֹתוֹ כִּי הָאִישׁ הַתָּמִים / אֵין לוֹ דָּמִים,</p></blockquote>
<blockquote><p>/ הֵחֵל לְהַלְאוֹתוֹ וּלְהַכְאִיבוֹ / וּלְעַנּוֹתוֹ וּלְהַרְעִיבוֹ. /</p></blockquote>
<blockquote><p>וְכִרְאוֹת הַנָּזִיר צָרָתוֹ וְחִילוֹ, / וַיֵּבְךְּ וַיִּתְחַנֶּן לוֹ /</p></blockquote>
<blockquote><p>וַיֹּאמַר ״לָמָּה תַעֲשֶׂה כֵן לְעַבְדֶּךָ / וְלָמָה אָמוּת נֶגְדֶּךָ? /</p></blockquote>
<blockquote><p>לֵךְ עִמִּי, אַל תֶּחְדַּל חָדוֹל – / וַאֲנִי אַעְשֶׁרְךָ עֹשֶׁר גָּדוֹל</p></blockquote>
<blockquote><p>/ וְאֶתֵּן לְךָ כֶּסֶף וְזָהָב / וְכֹל אֲשֶׁר תֶּאֱהַב״. / וַיֹּאמֶר לוֹ</p></blockquote>
<blockquote><p>״אִם כֵּן / שָׁאַתָּה עָשִׁיר וְלֹא מִסְכֵּן, / הוֹדִיעֵנִי נָא אֶת</p></blockquote>
<blockquote><p>דְּרָכֶיךָ / וּבַמֶּה נֶעְשַׁרְתָּ וּמַה מְּלַאכְתֶּךָ / וּמֵאַיִן</p></blockquote>
<blockquote><p>תָּבוֹא וּבְאֵיזֶה מָקוֹם יָשַׁבְתָּ / וּמָה אַרְצְךָ וְאֵי מִזֶּה עַם</p></blockquote>
<blockquote><p>אַתָּה״.</p> <p>וַיֹּאמַר ״אֲנִי הָיִיתִי מִתְעַסֵּק בִּסְחוֹרָה /</p></blockquote>
<blockquote><p>בְּצוֹר הַמַּעֲטִירָה, / וָאֱהִי רוֹכֵל בְּכָל מִינֵי תְמוּרָה /</p></blockquote>
<blockquote><p>בְּרֹאשׁ כָּל בֹּשֶׂם וּבְאֶבֶן יְקָרָה. / וַתִּגְדַּל מְאֹד יִפְעָתִי</p></blockquote>
<blockquote><p>וּתְבוּנַת מַחְשְׁבוֹתַי / וַתַּעַשׂ כֶּסֶף וְזָהָב בְּאוֹצְרוֹתַי. /</p></blockquote>
<blockquote><p>וַיִּבְעַט לִבִּי בְּיוֹצֵר אוֹר וּבוֹרֵא שַׁחַר / מֵרֹב כָּל הוֹן</p></blockquote>
<blockquote><p>בְּיֵין חֶלְבּוֹן וְצֶמֶר צַחַר, / וַיִּתְגַּבֵּר יִצְרִי עַל נַפְשִׁי</p></blockquote>
<blockquote><p>הַהֲדוּרָה / וַיְסִירֶהָ מִגְּבִירָה, / וַיֵּלֶךְ לוֹ לְבַקֵּשׁ גַּאֲוָה</p></blockquote>
<blockquote><p>/ אֶל קִבְרוֹת הַתַּאֲוָה. / וַיְהִי יוֹם אֶחָד בְּפָקְדִי עֲווֹנַי /</p></blockquote>
<blockquote><p>הוּא יוֹם יִוָּדַע לַייָ – / וְהִנֵּה כְאֵב גָּדוֹל – עַד לָמוּת</p></blockquote>
<blockquote><p>שָׁאֲלָה נַפְשִׁי / וַיִּקָּחֵנִי בְּצִיצִית רֹאשִׁי. / וָאֶשְׁלַח אֶל</p></blockquote>
<blockquote><p>הָרוֹפְאִים לִדְרֹשׁ / לְרַפּאוֹת כְּאֵב הָרֹאשׁ, / וָאַרְבֶּה לָהֶם</p></blockquote>
<blockquote><p>מֹהַר וּמַתָּן / בִּגְדֵי מֶשִׁי וְרִקְמָה וּשְׂרִיקוֹת פִּשְׁתָּן. /</p></blockquote>
<blockquote><p>וַיִּשְׁתַּדְּלוּ לַעֲשׂוֹת לִי רְפוּאָתָם, / אִישׁ אִישׁ כְּפִי</p></blockquote>
<blockquote><p>עֲבוֹדָתָם, / וַיְצַוּוּ עָלַי ״מִמַּאֲמָרֵינוּ אַל תִּגְרַע /</p></blockquote>
<blockquote><p>וְנִשְׁמַרְתָּ מִכָּל דָּבָר רַע – / וְהוּא כָל מַאֲכָל קָשֶׁה וְכָבֵד /</p></blockquote>
<blockquote><p>אֲשֶׁר לֹא יֵעָבֵד – / בְּקָנֶה וְקִדָּה לְהַשְׁלִים מַעֲלָתוֹ /</p></blockquote>
<blockquote><p>וְקִנְמָן-בֶּשֶׂם מַחֲצִיתוֹ״ / לְהִמָּלֵט מִדֶּבֶר וֶאָסוֹן / לִמְשֹׁחַ</p></blockquote>
<blockquote><p>רֹאשִׁי בְּשֶׁמֶן שָׂשׂוֹן. / וָאֶעֱשֶׂה כְּפִי מִצְוָתָם וְלֹא</p></blockquote>
<blockquote><p>אֵחַרְתִּי, / וּבְכָל אֵלֶּה נִשְׁמַרְתִּי וְנִמְשַׁחְתִּי – / וְלֹא</p></blockquote>
<blockquote><p>שָׁלַוְתִּי וְלֹא שָׁקַטְתִּי וְלֹא נַחְתִּי. / וָאֶשְׁלַח בְּעַד רוֹפֵא</p></blockquote>
<blockquote><p>אֶחָד מִגְּדוֹלֵיהֶם / שַׂר צִלָּם מֵעֲלֵיהֶם, / וְהִנֵּה הוּא בָקִי</p></blockquote>
<blockquote><p>בְכָל עִנְיָן / גָּדוֹל בְּחָכְמָה וּבְמִנְיָן. / וָאֹמְרָה אֵלָיו ״עַל</p></blockquote>
<blockquote><p>כָּל אֲשֶׁר הֵבִינוּ וְהִשְכִּילוּ – / כִּי לֹא יוֹעִילוּ וְלֹא</p></blockquote>
<blockquote><p>יַצִּילוּ. / וּמִיּוֹם עָשִׂיתִי מַאֲמָרָם – כִּמְעַט לֹא אָבַדְתִּי</p></blockquote>
<blockquote><p>אָבוֹד / כִּי הַכְּאֵב גָּדוֹל מְאֹד״. / וַיִּחַר אַף הָרוֹפֵא</p></blockquote>
<blockquote><p>וַיְרִיבֵנִי, / אַפּוֹ טָרָף וַיִּשְׂטְמֵנִי, / וַיֹּאמַר ״כָּל</p></blockquote>
<blockquote><p>הַמִּשְׁתַּדֵּל בִּרְפוּאַת פֶּתִי וְסָכָל / הַשְׁקֵט לֹא יוּכַל, / כִּי</p></blockquote>
<blockquote><p>לֹא יָבִין תַּבְנִית הַמַּרְפֵּא בְּעִנְיָנוֹ – / לֹא יָדַע שָׁלֵו</p></blockquote>
<blockquote><p>בְּבִטְנוֹ. / וְהִנְנִי אוֹדִיעֲךָ שֹׁרֶשׁ הָרְפוּאָה וְעִקָּרֶיהָ / אַף</p></blockquote>
<blockquote><p>הִיא תָּשִׁיב אֲמָרֶיהָ…</p> <p>וַיְהִי בְשָׁמְעִי דִבְרֵי הָרוֹפֵא וְחָכְמָתוֹ /</p></blockquote>
<blockquote><p>וְהִתְבּוֹנְנִי בְּהַנְהָגָתוֹ וּתְבוּנָתוֹ, / וָאֶדֹּר עַל פְּשָׁעַי</p></blockquote>
<blockquote><p>וַעֲווֹנַי / נֶדֶר נָזִיר לְהַזִּיר לַייָ, / וָאֲחַפֵּשׂ מַעֲשַׂי /</p></blockquote>
<blockquote><p>וְהִקְדַּשְׁתִּי כָל נְכָסַי. / וַיַּעַשׂ לִי הָרוֹפֵא רְפוּאוֹת /</p></blockquote>
<blockquote><p>לְמוֹפֵת וּלְאוֹת / וַיִּשְׁלַח אֱלֹהִים מַרְגִּיעַ / מַלְאָךְ מֵלִיץ</p></blockquote>
<blockquote><p>וּמוֹשִׁיעַ, / וַיִּתְחַזֵּק הַטֶּבַע בְּלִבִּי / וַיֵּלֶךְ הַמִּקְרֶה</p></blockquote>
<blockquote><p>מִקִּרְבִּי / וַיָּנָס הַכְּאֵב וְהַנֵּזֶק מֵעַל נַפְשִׁי, / וָאָסִיר</p></blockquote>
<blockquote><p>עֲטֶרֶת רֹאשִׁי. / וָאָקוּם וָאֶבְנֶה בַיִת מְקֻדָּשׁ / לֵאלֹהִים מְעַט</p></blockquote>
<blockquote><p>מִקְדָּשׁ, / וָאָכִין אוֹתוֹ מִקֶּדֶם לָעִיר / לִקְרֹא לְכָל עוֹבֵר</p></blockquote>
<blockquote><p>וָשָׁב וּלְהָעִיר. / וָאֲחַבֵּר אֵלַי חֲבֵרִים וְתַלְמִידִים / רַבִּים</p></blockquote>
<blockquote><p>וְנִכְבָּדִים, / וָאֶמְאַס סַלּוֹנִים וְסָרָבִים / וָאַרְבִּיץ תּוֹרָה</p></blockquote>
<blockquote><p>בָּרַבִּים. / וָאֶתֵּן לְכָל אִישׁ כִּפְרִי מַעֲלָלָיו / בַּבַּיִת</p></blockquote>
<blockquote><p>אֲשֶׁר נִקְרָא שְׁמִי עָלָיו, / וָאַרְבֶּה בְתוֹכוֹ עֹשֶׁר וְכָבוֹד /</p></blockquote>
<blockquote><p>וְכֶסֶף וְזָהָב הַרְבֵּה מְאֹד, / וְהַכֹּל מוּכָן לַעֲנִיִּים /</p></blockquote>
<blockquote><p>וּלְפִדְיוֹן שְׁבוּיִים.</p> <p>וַיְהִי הַיּוֹם וַאֲנִי יוֹשֵׁב עַל מְכוֹנִי /</p></blockquote>
<blockquote><p>שָׁמַעְתִּי וַתִּרְגַּז בִּטְנִי – / וְהִנֵּה אִישׁ מִבְּנֵי עַמֵּנוּ</p></blockquote>
<blockquote><p>נֶחְבָּא / וּבֵין הַגּוֹיִים נִשְׁבָּה. / וָאֵלֵךְ הַיּוֹם לִפְדּוֹתוֹ /</p></blockquote>
<blockquote><p>וּלְהָשִׁיב אֶת שְׁבוּתוֹ / וְהִנֵּה שַׁבְתִּי לְדַרְכִּי / לָלֶכֶת</p></blockquote>
<blockquote><p>לְבֵיתִי וְאֶל קְהִלַּת פִּלְכִּי. / וְעַתָּה מַהֵר לֵךְ עִמִּי / אֶל</p></blockquote>
<blockquote><p>בֵּיתִי וְאוּלַמִּי / וְאַרְבֶּה לְךָ מֹהַר וּמַתָּן לְהַשְׁכִּיחַ</p></blockquote>
<blockquote><p>רִישֶׁךָ / כְּכֹל אֲשֶׁר תִּשְׁאַל נַפְשְׁךָ״. / וַיֹּאמֶר הָאִישׁ</p></blockquote>
<blockquote><p>״אֵיךְ אֵלֶךְ? אוּלַי יִתְחַבְּרוּ תַלְמִידֶיךָ / וְאַתָּה וַעֲבָדֶיךָ /</p></blockquote>
<blockquote><p>וְיוֹדִיעוּ לַמֶּלֶךְ דְּבַר אִישׁ כָּמוֹנִי / וְיֹאחֲזוּנִי יְמֵי</p></blockquote>
<blockquote><p>עֹנִי. / וְעַתָּה הִשָּׁבְעָה לִי בְּצוּר מָעֻזִּי וְקוֹנִי / אִם</p></blockquote>
<blockquote><p>תִּשְׁקֹר לִי וּלְנִינִי!״ / וַיֹּאמֶר הַנָּזִיר ״אָנֹכִי אִשָּׁבַע, /</p></blockquote>
<blockquote><p>שָׂפָה לְפִי סָבִיב לֹא יִקָּרֵעַ״.</p> <br/></blockquote>
<blockquote><p>וַיָּקוּמוּ מִשָּׁם הָאֲנָשִׁים / וַיֵּלְכוּ הָלוֹךְ</p></blockquote>
<blockquote><p>וְהֵמָּה מַחְשִׁים / וַיָּגִיעוּ אֶל בֵּית הַוַּעַד / בְּאֵימָה וָרַעַד</p></blockquote>
<blockquote><p>/ וְכָל הַתַּלְמִידִים יוֹשְׁבִים בִּמְסִבָּם / יָסֹבוּ עַל דַרְכָּם</p></blockquote>
<blockquote><p>וּנְתִיבָם. / וַיִּשְׁתַּחֲווּ לוֹ וַיִּשְׂמְחוּ לִקְרָאתוֹ /</p></blockquote>
<blockquote><p>וַיֵּשְׁבוּ לְפָנָיו הַבְּכוֹר כִּבְכוֹרָתוֹ. / וַיִּשְׁאַל הַנָּזִיר</p></blockquote>
<blockquote><p>בִּשְׁלוֹמָם וּבִשְׁלוֹם תּוֹרָתָם / וּשְׁלוֹם תַּלְמוּדָם וּמִשְׁנָתָם.</p></blockquote>
<blockquote><p>/ וַיֹּאמְרוּ ״שָׁלוֹם – תְּהִלָּה לְשׁוֹכֵן מְעוֹנַי! – / בָּרוּךְ</p></blockquote>
<blockquote><p>אַתָּה לַייָ. / אָמְנָם נִסְתַּפֵּק עָלֵינוּ הֲלָכָה וְנִשְׁאֲרָה</p></blockquote>
<blockquote><p>לְפָרֵשׁ / אַרְבָּעָה חֲלוּקֵי כַפָּרָה שֶׁהָיָה רַבִּי יִשְׁמָעֵאל</p></blockquote>
<blockquote><p>דּוֹרֵשׁ״. / אָמַר לָהֶם ״שְׁלֹשָׁה הֵם / וּתְשׁוּבָה עַל כָּל אֶחָד מֵהֶם״…</p> <p>וַיִּשְׂמְחוּ הַתַּלְמִידִים בְּמִדְרָשׁוֹ / וְשׂוֹם</p></blockquote>
<blockquote><p>שֶׂכֶל וְהָבֵן בְּפֵרוּשׁוֹ / וַיֹּאמְרוּ לוֹ ״שָׁלוֹם לְךָ וְשָׁלוֹם</p></blockquote>
<blockquote><p>לִדְבָרֶיךָ הַנֶּחְמָדִים / אֲשֶׁר שַׂמְתָּ שָׁלוֹם בֵּין</p></blockquote>
<blockquote><p>הַתַּלְמִידִים!״</p> <br/></blockquote>
<blockquote><p>וַיְהִי בְהִתְבּוֹנֵן הָאִישׁ בִּשְׁאֵלַת הַתַּלְמִידִים</p></blockquote>
<blockquote><p>/ וַיְהִי כִּרְאוֹת אֶת הַנֶּזֶם וְאֶת הַצְּמִידִים / וּבְשָׁמְעוֹ אֶת</p></blockquote>
<blockquote><p>דִּבְרֵי הַנָּזִיר וּפֵרוּשׁוֹ – / וַיִּשְׁאַל בְּאָלָה נַפְשׁוֹ /</p></blockquote>
<blockquote><p>וַיֹּאמֶר בְּלִבּוֹ: אֲהָהּ כִּי נִדְמֵיתִי בְאַשְׁמָתִי, / אוֹי לִי עַל</p></blockquote>
<blockquote><p>שִׁבְרִי, נַחְלָה מַכָּתִי! / אַלְלַי לִי כִּי בָא קָלוֹן / וַיָּבוֹא</p></blockquote>
<blockquote><p>זָדוֹן! / אוֹי לִי, אִמִּ, כִּי יְלִדְתִּנִי אִישׁ רִיב וְאִישׁ מָדוֹן –</p></blockquote>
<blockquote><p>/ וְלֹא שָׁמַעְתִּי לְקוֹל הוֹרַי / וְלֹא הִטִּיתִי אֹזֶן לִמְלַמְּדַי</p></blockquote>
<blockquote><p>וּמוֹרַי! / וּמָה אֶעֱשֶׂה כִּי יָקוּם אֵל וְכִי יִפְקֹד / וְהָעֵת</p></blockquote>
<blockquote><p>אֲשֶׁר עַל הָרָעָה יִשְׁקֹד? / וּמָה אֲשִׁיבֶנּוּ עֵת יְמַהֵר עַל</p></blockquote>
<blockquote><p>הָאֲדָמָה לְהַבְקִיעֶנָּה / לֶאֱחֹז בְּכַנְפֵי הָאָרֶץ וְיִנָּעֲרוּ</p></blockquote>
<blockquote><p>רְשָׁעִים מִמֶּנָּה? / לָמָּה הָיָה כְאֵבִי נֶצַח וּמַכָּתִי אֲנוּשָׁה /</p></blockquote>
<blockquote><p>מִקֹּצֶר רוּחַ וּמֵעֲבוֹדָה קָשָׁה?</p> <p>וַיְהִי הוּא טֶרֶם כִּלָּה לְהַרְהֵר מַחְשְׁבוֹתָיו /</p></blockquote>
<blockquote><p>בְּגֹדֶל אֲשָׁמָיו / וְרֹעַ חַטֹּאותָיו – / וְהִנֵּה קוֹל דְּמָמָה בָאָה</p></blockquote>
<blockquote><p>לְבַשְּׂרוֹ / וַתְּסַמֵּר שַׂעֲרַת בְּשָׂרוֹ / וַתֹּאמַר ״שׁוּבוּ</p></blockquote>
<blockquote><p>בָּנִים שׁוֹבָבִים / הָרְחוֹקִים וְהַקְּרוֹבִים, / כִּי לֹא אֶחְפֹּץ</p></blockquote>
<blockquote><p>בְּמוֹת הָרָשָׁע נֶחְפָּר וְנִכְלָם, / כִּי חָסִיד אֲנִי, נְאֻם יְיָ,</p></blockquote>
<blockquote><p>לֹא אֶטֹּר לְעוֹלָם!״ / וַיִּתֵּן הָאִישׁ אֶת קוֹלוֹ בִצְעָקָה /</p></blockquote>
<blockquote><p>בְּשָׁמְעוֹ קוֹל דְּמָמָה דַקָּה. / וַיִּשְׁמַע הַנָּזִיר אֶת קוֹלוֹ /</p></blockquote>
<blockquote><p>וַיַּרְא וְהִנֵּה הוּא מְסַלֵּד בְּחִילוֹ. / וַיֹּאמַר ״מַה לְּךָ כִּי</p></blockquote>
<blockquote><p>נִזְעַקְתָּ, / הֲרִימֹתָ קוֹלְךָ וְצָעַקְתָּ? / אוּלַי יָרֵאתָ</p></blockquote>
<blockquote><p>מִתַּלְמִידַי נֶאֱמָנַי / הָעוֹמְדִים שָׁם לְשָׁרֵת אֶת פְּנֵי יְיָ? /</p></blockquote>
<blockquote><p>חֲזַק וֶאֱמַץ, אַל תִּירָא, / כִּי אִתְּךָ אֲנִי לְצוּר מָעֹז</p></blockquote>
<blockquote><p>וּלְעֶזְרָה! / וְהִנְנִי נוֹתֵן לְךָ כָּעֵת מָחָר / כֶסֶף וְזָהָב כְּכֹל</p></blockquote>
<blockquote><p>אֲשֶׁר תִּבְחַר, / וְאֶשְׁלָחֲךָ בְּשִׁירִים וּבְשִׂמְחָה / בְּהַשְׁקֵט</p></blockquote>
<blockquote><p>וּבְבִטְחָה, / וְלֹא אֲגַלֶּה סוֹדְךָ הַנֶּעְלָם / אֲשֶׁר נִשְׁבַּעְנוּ</p></blockquote>
<blockquote><p>שְׁנֵינוּ בְּשֵׁם יְיָ אֵל עוֹלָם״.</p> <p>וַיֹּאמֶר הָאִישׁ ״עַל זֶה לֹא נִהְיֵיתִי וְנֶחֱלֵיתִי,</p></blockquote>
<blockquote><p>/ לֹא צָעַקְתִּי כִּי יָרֵאתִי – / וְיִרְאָתִי קָשָׁה וַעֲצוּמָה /</p></blockquote>
<blockquote><p>כִּמְעַט שָׁכְנָה נַפְשִׁי דוּמָה: / כִּי אֵין בִּי כֹחַ לְהַגִּיד</p></blockquote>
<blockquote><p>עֲווֹנוֹתַי, / עָצְמוּ מִסַּפֵּר תְּלָאוֹתַי. / כִּי בְעָווֹן</p></blockquote>
<blockquote><p>חוֹלַלְתִּי / וּבִדְמֵי חַטָּאתִי הִתְגּוֹלַלְתִּי, / אָשַׁמְתִּי</p></blockquote>
<blockquote><p>בָּגַדְתִּי גָּזַלְתִּי / וְהֶבֶל נָחַלְתִּי, / וְעַל כֵּן חַלְתִּי</p></blockquote>
<blockquote><p>וְהִתְחַלְחַלְתִּי / וּבֶעָפָר קַרְנִי עוֹלַלְתִּי: / כִּי אֲשָׁמַי</p></blockquote>
<blockquote><p>הִדְאִיבוּ כֹחִי / וְחַטֹּאותַי הִרְפּוּ מֹחִי / אֲשֶׁר חֲמָתָם שׁוֹתָה</p></blockquote>
<blockquote><p>רוּחִי, / וַעֲווֹנִי מִהַר לְהַכְלִימֵנִי, / וְתוּשִׁיָּה נִדְּחָה</p></blockquote>
<blockquote><p>מִמֶּנִּי. / וְעַתָּה, אִם יֶשׁ לְךָ עַל זֶה עֵצָה / יוֹם עֶבְרָה</p></blockquote>
<blockquote><p>וּנְאָצָה / וְאִם יָדַעְתָּ רְאִיָּה וּמַחֲזֶה – / הַאֶחְיֶה מֵחֳלִי</p></blockquote>
<blockquote><p>זֶה?״</p> <br/></blockquote>
<blockquote><p>וַיֹּאמֶר הַנָּזִיר ״אִם תָּשׁוּב אֶל יְיָ, תִּזְכֶּה</p></blockquote>
<blockquote><p>וְתִחְיֶה / לְמַעְלָה בַּזְּמָן תִּהְיֶה / וּבְצֵל אַרְבָּעָה דְבָרִים</p></blockquote>
<blockquote><p>הַלָּלוּ תִּתְלוֹנֵן וְתֶחְסֶה: / צְעָקָה, צְדָקָה, שִׁנּוּי הַשֵּׁם,</p></blockquote>
<blockquote><p>שִׁנּוּי מַעֲשֶׂה, / וְאַף שִׁנּוּי מָקוֹם תּוֹעֶלֶת גְּדוֹלָה / כְּלֵי</p></blockquote>
<blockquote><p>גוֹלָה״. / וַיֹּאמֶר הָאִישׁ ״הִנֵּה עֲוֹנִי מִנְּשֹׂא גָדַל / וְלִבִּי</p></blockquote>
<blockquote><p>מֵהָבִין מָאַס וְחָדַל / וְאֵיךְ יִתְכַּפֵּר בְּשִׁנּוּיִים, / דְּבָרִים</p></blockquote>
<blockquote><p>קַלִּים מְצוּיִים, / צְדָקָה וּצְעָקָה וְשִׂיחָה / אוֹ בְזֶבַח</p></blockquote>
<blockquote><p>וּבְמִנְחָה?״ / וַיֹּאמֶר הַנָּזִיר ״אִם בְּזֶבַח וּמִנְחָה אֵין לוֹ</p></blockquote>
<blockquote><p>כַּפָּרָה, / מִתְכַּפֵּר בְּדִבְרֵי תוֹרָה״. / וַיֹּאמֶר הָאִישׁ</p></blockquote>
<blockquote><p>״הִנְנִי מַשְׁבִּיעֲךָ בְּשֵׁם הָאֵל הַנֶּעְלָם / שֶׁכּוֹחוֹ מָלֵא</p></blockquote>
<blockquote><p>עוֹלָם / שְׁתְּיָעֲצֵנִי עֵצָה נְכוֹנָה / בְּדַעַת וּבִתְבוּנָה: / אֵיךְ</p></blockquote>
<blockquote><p>אוּכַל לְהָרִים רֹאשִׁי / וּלְהַכְנִיעַ גַּאֲוָתִי וְכַחֲשִׁי – /</p></blockquote>
<blockquote><p>וְתַגְדֵּל חַסְדְּךָ אֲשֶׁר עָשִׂיתָ עִמָּדִי לְהַחֲיוֹת אֶת נַפְשִׁי. /</p></blockquote>
<blockquote><p>בִּשְׁבוּעָה אֲקַבֵּל דְּבָרֶיךָ וַעֲצָתְךָ / וְאָבוֹא בִּבְרִית</p></blockquote>
<blockquote><p>אִתְּךָ״.</p> <p>וַיֹּאמֶר הַנָּזִיר ״אִם כֵּן – הִדָּבֵק בַּתְּשׁוּבָה /</p></blockquote>
<blockquote><p>וְלֵךְ בְּדֶרֶךְ חֲשׁוּבָה / וּמְאֹד מְאֹד הֱוֵי שְׁפַל רוּחַ /</p></blockquote>
<blockquote><p>וְתִשְׁקֹט וְתָנוּחַ, / וּרְדֹף אַחֲרֵי הַצְּדָקָה הַחֲמוּדָה /</p></blockquote>
<blockquote><p>וְשָׁמַרְתָּ אֶת הַחֻקָּה הַזֹּאת בְּמוֹעֲדָהּ. / וְאִם תִּרְאֶה רָעֵב</p></blockquote>
<blockquote><p>וְצָמֵא / חָרֵד וְהוֹמֶה – / תְּמַהֵר לְהַאֲכִילוֹ ולְהַשְׁקוֹתוֹ, /</p></blockquote>
<blockquote><p>וְכִי תִרְאֶה עָרֹם וְכִסִּיתוֹ. / וְאֵין בְּזוֹ הַמִּצְוָה שִׁעוּר</p></blockquote>
<blockquote><p>יָדוּעַ / עַל כָּל אָדָם קָבוּעַ / אָמְנָם הַכֹּל כְּפִי הוֹנוֹ</p></blockquote>
<blockquote><p>וּכְבוֹדוֹ, / אִישׁ כְּמַתְּנַת יָדוֹ. / וַאֲפִלּוּ אֵין לְאֵל בְּיַד</p></blockquote>
<blockquote><p>מְמוֹנֶךָ / לַעֲשׂוֹת כְּפִי חֶפְצְךָ וּרְצוֹנֶךָ – / הֱיֵה מְפַיֵּס</p></blockquote>
<blockquote><p>לַדַּל וְדוֹבֵר עַל לְבָבוֹ / וְנוֹתֵן הוֹדָאָה לְשֶׁעָבַר וְצוֹעֵק</p></blockquote>
<blockquote><p>לֶעָתִיד לָבוֹא: / זוֹ הִיא צְעָקָה וּצְדָקָה גְּדוֹלָה / עַל כֹּל</p></blockquote>
<blockquote><p>עוֹלָה״. / וַיֹּאמֶר הָאִישׁ ״הֲרֵינִי מְקַבֵּל כֹּל אֲשֶׁר תֹּאמַר, /</p></blockquote>
<blockquote><p>הוֹשִׁיעֵנִי, אֲדוֹנִי, כִּי גָמַר״. / וַיֹּאמֶר הַנָּזִיר ״מַה שִּׁמְךָ</p></blockquote>
<blockquote><p>הָאֶזְרָחִי?״ / וַיֹּאמַר ״חֶבֶר הָאֲחוֹחִי״. / וַיֹּאמַר ״חָבֵר שִׁמְךָ</p></blockquote>
<blockquote><p>יִקָּרֵא / וְיִהְיֶה עַל תְּשׁוּבָתְךָ מוֹרֶה, / וּשְׂנָא חֶבְרַת</p></blockquote>
<blockquote><p>הַזֵּדִים וְהַגֵּאִים / יִתַּמּוּ חַטָּאִים, / יִהְיוּ בִלְבָבְךָ כָּל</p></blockquote>
<blockquote><p>עוֹשֵׂי רִשְׁעָה לְמַכְאוֹב / וְחוֹבֵר חֶבֶר וְשׁוֹאֵל אוֹב, / וְשַׁנֵּה</p></blockquote>
<blockquote><p>אֶת שִׁמְךָ כַּאֲשֶׁר אֲדוֹנִי דּוֹבֵר, / אָהוּב וְרֵעַ וְחָבֵר. /</p></blockquote>
<blockquote><p>וְשַׁבְתָּ וְרָאִיתָ בְּטוּב שַׁדַּי וְהוֹדוֹ / בֵּין עוֹבֵד לֵאלֹהִים</p></blockquote>
<blockquote><p>אֲשֶׁר לֹא עֲבָדוֹ. / וְהָעֵד עָלֶיךָ לַצּוּר שׁוֹכֵן מְרוֹמִים / וְעַל</p></blockquote>
<blockquote><p>כָּל פְּעָלָיו הַנְּעִימִים, / וֶאֱמֹר לְפָנָיו בְּכָל הַדְּבָרִים</p></blockquote>
<blockquote><p>אֲשֶׁר יִמְצָאוּךָ: / חָבֵר אֲנִי לְכָל אֲשֶׁר יְרֵאוּךָ! / זֶהוּ</p></blockquote>
<blockquote><p>שִׁנּוּי הַשֵּׁם וְשִׁנּוּי הַמַּעֲשֶׂה / וְתַחַת כְּנָפָיו תֶּחְסֶה. /</p></blockquote>
<blockquote><p>וְשִׁנּוּי מָקוֹם כְּלָלִים וּפְרָטִים – / מַעְיָן מִבֵּית יְיָ יֵצֵא</p></blockquote>
<blockquote><p>וְהִשְׁקָה אֶת נַחַל הַשּׁוֹטִים: / כִּי דֶרֶךְ רֹב בְּנֵי אָדָם /</p></blockquote>
<blockquote><p>שׁוֹכְנֵי בָתֵי חֹמֶר אֲשֶׁר בֶּעָפָר יְסוֹדָם / לִהְיוֹת בַּדַּעַת</p></blockquote>
<blockquote><p>נִמְשָׁכִים / וּכְמִנְהַג מְדִינוֹת מוֹשָׁבָם הוֹלְכִים. / עַל כֵּן</p></blockquote>
<blockquote><p>צָרִיךְ אָדָם לְהִתְחַבֵּר לַחֲכָמִים / וְלָשֶׁבֶת בְּמוֹשַׁב</p></blockquote>
<blockquote><p>הַתְּמִימִים, / וְיִתְרַחֵק מִן הָרְשָׁעִים / וּמֵחֲבוּרַת מְרֵעִים. /</p></blockquote>
<blockquote><p>וְאִם הָיָה בִמְדִינָה שֶׁיּוֹשְׁבֶיהָ מַמְרִים / וְשָׂרֶיהָ סוֹרֲרִים /</p></blockquote>
<blockquote><p>וְאֵינָם מִתְנַהֲגִים כַּשּׁוּרָה / עַל פִּי הַתּוֹרָה – / יָשׁוּב אֶל</p></blockquote>
<p>< נַפְשׁוֹ וִיעִירֶהָ / וּמִצִּנִּים וּפַחִים יְסִירֶהָ / בְאָמְרוֹ:</p>
<blockquote><p>רְאִי, נַפְשִׁי, דַּרְכֵי הֶהָמוֹן / בְּתֹהוּ יְלֵל יְשִׁימוֹן / כַּמָּה</p></blockquote>
<blockquote><p>מַעֲשֵׂיהֶם מְכֹעָרִים / זְקֵנִים בַּחוּרִים וּנְעָרִים / אֵיךְ</p></blockquote>
<blockquote><p>בְּמִנְהָגָם כָּל תָּם וְיָשָׁר יַזִּיקוּ / וְיַד תַּלְמִיד חָכָם לֹא</p></blockquote>
<blockquote><p>יַחֲזִיקוּ / וּכְסִילִים שָׂמוּ לְשָׂרַיךְ וּלְשׁוֹפְטַיִךְ / וְזֵדִים</p></blockquote>
<blockquote><p>וְהוֹלֲלִים חִזְּקוּ לִשְׁפֹּט שְׁפָטַיִךְ / אַלּוֹנִים מִבָּשָׁן עָשׂוּ</p></blockquote>
<blockquote><p>מִשּׁוֹטַיִךְ! / וְעַתָּה, נַפְשִׁי, לְכִי וְנֵלְכָה בְּאוֹר אֱלֹהֵינוּ</p></blockquote>
<blockquote><p>/ לְהַרְבּוֹת מַהֲלָלֵנוּ וְנָגְהֵנוּ / אֶל מְקוֹם אֲנָשִׁים צַדִּיקִים</p></blockquote>
<blockquote><p>/ וּבַתּוֹרָה מַחֲזִיקִים, / וְנִתְחַבֵּר לָהֶם בְּכָל מִינֵי חִבּוּר /</p></blockquote>
<blockquote><p>לָדַעַת וְלָבוּר – / וְאִם אֵינוֹ יָכוֹל לָלֶכֶת / מִפְּנֵי מַחֲלָה</p></blockquote>
<blockquote><p>וְאָלָה נִתֶּכֶת / אוֹ מִפְּנֵי אֻמָּה אַדִּירָה אוֹ מַמְלֶכֶת / לַהַט</p></blockquote>
<blockquote><p>הַחֶרֶב הַמִּתְהַפֶּכֶת – / יִתְנַהֵג יְחִידִי וְיִתְבּוֹנֵן</p></blockquote>
<blockquote><p>בְּמִפְעָלָיו / יֵשֵׁב בָּדָד וְיִדֹּם כִּי נָטַל עָלָיו. / וְאִם</p></blockquote>
<blockquote><p>לְבַדּוֹ לֹא יַנִּיחוּהוּ – יֵצֵא לְמִדְבָּרוֹת וּלְצִיּוֹת / לְהָרִים</p></blockquote>
<blockquote><p>וְלִגְבָעוֹת / לַאֲפִיקִים וּלְגֵיאָיוֹת, / וְיִתְנַהֵג בַּעֲנָוָה / עַל</p></blockquote>
<blockquote><p>פִּי הַמִּצְוָה / וְיִסְתַּכֵּל בְּמַעֲשֵׂה הַנִּשָּׂא וְהַגָּבֹהַּ /</p></blockquote>
<blockquote><p>וּמִבְּשָׂרוֹ יֶחֱזֶה אֱלוֹהַּ. / אֵלֶּה הֵם רָאשֵׁי מִדּוֹת הַשָּׁב</p></blockquote>
<blockquote><p>וּכְלָלָם – / וְתַלְמוּד תּוֹרָה כְּנֶגֶד כֻּלָּם!״</p> <br/></blockquote>
<blockquote><p>וַיֹּאמֶר הָאִישׁ ״אַחֲרֵי אֲשֶׁר מִצּוּף דְּבָרֶיךָ</p></blockquote>
<blockquote><p>הִשְׂבַּעְתַּנִי / וּמַעֲלוֹת מִדּוֹת טוֹבוֹת הוֹדַעְתַנִי, / הִנְנִי</p></blockquote>
<blockquote><p>מְקַיֵּם אִמְרוֹתֶיךָ הַצְּרוּפוֹת / בְּסֵבֶר פָּנִים יָפוֹת / וְכָל</p></blockquote>
<blockquote><p>יְמֵי חַיַּי אֶעֱבֹר לְפָנֶיךָ / כֹּה יַעֲשֶׂה לִי אֱלֹהִים וְכֹה</p></blockquote>
<blockquote><p>יוֹסִיף, כִּי הַמָּוֶת יַפְרִיד בֵּינִי וּבֵינֶךָ!״</p> <p>וַיִּקָּחֵהוּ וַיְבִיאֵהוּ אֶל בֵּית רְפִידָתוֹ</p></blockquote>
<blockquote><p>הַנֶּאֱהָב / אֲשֶׁר שָׁם הַזָּהָב / וַיִּתֶּן לוֹ כִּכַּר כֶסֶף</p></blockquote>
<blockquote><p>בְּמַתָּנָה / וַחֲמֵשׁ חֲלִיפוֹת שְׂמָלוֹת לוֹ לְמָנָּה / וּמִזְּהַב</p></blockquote>
<blockquote><p>שְׁבָא לִטְרָא אַחַת מְזֻקֶּקֶת / טוּר אֹדֶם פִּטְדָה וּבָרֶקֶת. /</p></blockquote>
<blockquote><p>וַיֹּאמֶר לוֹ ״לֵךְ וְהַעֲבֵר קוֹל בְּכָל מְדִינוֹת הַנּוֹשָׁבוֹת /</p></blockquote>
<blockquote><p>בַּשְּׁוָקִים וּבָרְחוֹבוֹת / וְשַׁלֵּם לְכָל עָשׁוּק מָמוֹן וְאִמְרֵי</p></blockquote>
<blockquote><p>שֶׁפֶר / אֶת מִי עַשַׁקְתָּ אֶת מִי רַצּוֹתָ וּמִיַּד מִי לָקַחְתָּ</p></blockquote>
<blockquote><p>כֹפֶר, / וְאַל יִרַע לְבָבְךָ בְּתִתְּךָ לוֹ מְמוֹנו וְכֵלָיו /</p></blockquote>
<blockquote><p>וַחֲמִישִׁיתוֹ הוֹסֵף עָלָיו״.</p> <p>וַיַּעֲבֹר הָאִישׁ וַיַּעַשׂ כְּכָל אֲשֶׁר צֻוָּה, /</p></blockquote>
<blockquote><p>כִּי מִדְרָשׁוֹ לְמוֹשָׁב לוֹ אִוָּה, / וַיְדַבֵּר עַל כָּל לֵב עָשׁוּק</p></blockquote>
<blockquote><p>וְגָזוּל עַד אֲשֶׁר נִתְרַצָּה, לְהָשִּׁיב אֶת הַגְּזֵלָה אֲשֶׁר גָּזַל</p></blockquote>
<blockquote><p>אוֹ אֶת הָעֹשֶׁק אֲשֶׁר עָשַׁק אוֹ אֶת הַפִּקָּדוֹן אֲשֶׁר הָפְקַד</p></blockquote>
<blockquote><p>אִתּוֹ אוֹ אֶת הָאֲבֵדָה אֲשֶׁר מָצָא, / וַיִּתֵּן לַעֲנִיֵּי</p></blockquote>
<blockquote><p>הַמְּדִינוֹת צְדָקָה / וַיֶּרֶב עַל חֶטְאוֹ תְּפִלָּה וּצְעָקָה /</p></blockquote>
<blockquote><p>וַיְשַׁנֶּה מַעֲשָׂיו לְטוֹבָה / כְּמִנְהַג כָּל בַּעַל תְּשׁוּבָה /</p></blockquote>
<blockquote><p>וַיָּעַד עַל פְּעָלָיו הַטּוֹבִים שְׁמוֹ / וַיְשַׁנֶּה אֶת טַעְמוֹ /</p></blockquote>
<blockquote><p>וַיִּלְבַּשׁ שְׁחוֹרִים וַיִּתְכַּס / וְגֹבַהּ לֵב שָׂם לְמִרְמָס, /</p></blockquote>
<blockquote><p>וַיֵּשֶׁב בְּבֵית הַנָּזִיר בְּאַהֲבָה / גָּרְסָה נַפְשׁוֹ לְתַאֲבָה. /</p></blockquote>
<blockquote><p>וְכִרְאוֹת הַנָּזִיר שִׁנּוּי פְּעָלָיו וּמַעֲשֵׂהוּ / וַיָּרַח אֶת</p></blockquote>
<blockquote><p>רֵיחַ בְּגָדָיו וַיְבָרְכֵהוּ / וַיֹּאמַר ״תִּחְיֶה וְתִזְכֶּה בַתּוֹרָה</p></blockquote>
<blockquote><p>כְּפִי חֶפְצֶךָ, / יְיָ אֱלֹהֶיךָ יִרְצֶךָ!״ / וַיְהִי מִמָּחֳרָת</p></blockquote>
<blockquote><p>וַיֵּשֶׁב הַנָּזִיר לְלַמְּדוֹ תוֹרָה / וַיִּלְמַד כָּל אֲשֶׁר הוֹרָה: /</p></blockquote>
<blockquote><p>תּוֹרָה נְבִיאִים כְּתוּבִים בְּיַחַד, / שְׁלֹשֶׁת הָרוֹעִים בְּיֶרַח</p></blockquote>
<blockquote><p>אֶחָד. / וַיִּלְמַד פֵּרוּשִׁים וְדִקְדּוּקִים, / חֻקִּים וּמִשְׁפָּטִים</p></blockquote>
<blockquote><p>צַדִּיקִים, / מִקְרָא מִשְׁנָה תַּלְמוּד וַאֲגַדְתָּא, / סִפְרֵי סִפְרָא</p></blockquote>
<blockquote><p>וְתוֹסֶפְתָּא. / וַיִּגְדַּל הָאִישׁ בְּמַעֲלַת הָעֹשֶׁר וְהַכָּבוֹד /</p></blockquote>
<blockquote><p>וַיֵּלֶךְ הָלוֹךְ וְגָדֵל עַד כִּי גָדַל מְאֹד. / וַיְשִׂימֵהוּ</p></blockquote>
<blockquote><p>הַנָּזִיר עַל תַּלְמִידָיו נָגִיד וּמְצַוֶּה / מַחְמִיר וּמֵקֵל</p></blockquote>
<blockquote><p>וּמַשְׁוֶה, / כִּי כָל דְּבָרָיו עַל פִּי הַתּוֹרָה שְׁקָלָן וַחֲלָקָן /</p></blockquote>
<blockquote><p>חֲזָקָה עַל חָבֵר שֶׁאֵינוֹ מוֹצִיא מִתַּחַת יָדוֹ דָּבָר שֶׁאֵנוֹ</p></blockquote>
<blockquote><p>מְתֻקָּן.</p> <p>וַיִּשְׁמַע בִּלְדַּד אָבִיו עִנְיָנוֹ / וַיִּיטַב</p></blockquote>
<blockquote><p>הַדָּבָר בְּעֵינָיו עַל אוֹדוֹת בְּנוֹ / וַיִּשְׁלַח אֵלָיו סְפָרִים</p></blockquote>
<blockquote><p>וַאֲרֻחָה / מִנְחָה הִיא אֶל הַנָּזִיר שְׁלוּחָה / כֶסֶף וְזָהָב</p></blockquote>
<blockquote><p>וּבְגָדִים כַּאֲשֶׁר יֵאוֹת וְקִדָּה חֲמֵשׁ מֵאוֹת. / וַיִּתֶּן לוֹ</p></blockquote>
<blockquote><p>הַנָּזִיר בִּתּוֹ לְאִשָּׁה – / תְּהִלָּה תִפְאֶרֶת קְדֻשָּׁה – / כִּי</p></blockquote>
<blockquote><p>אָמַר: לְעוֹלָם יִמְכֹּר אָדָם כָּל אֲשֶׁר לוֹ / לְהַרְבּוֹת מַהֲלָלוֹ,</p></blockquote>
<blockquote><p>/ וְיַשִּׂיא בִתּוֹ לְתַלְמִיד חָכָם / וְיָבִין וְיֶחְכַּם, כִּי בַת</p></blockquote>
<blockquote><p>הֶחָכָם הַנְּשׂוּאָה לְתַלְמִיד / עוֹלָה תָמִיד: / כִּי הוּא דָבָר</p></blockquote>
<blockquote><p>מִתְקַבֵּל וְנָאֶה / יַעֲלֶה וְיָבוֹא יַגִּיעַ יֵרָאֶה, / מָשָׁל</p></blockquote>
<blockquote><p>לְעִנְבֵי הַגֶּפֶן בַּגֶּפֶן, / וְאֶל אָוֶן נַפְשׁוֹ אַל תֵּפֶן. / כִּי</p></blockquote>
<blockquote><p>הַתַּלְמִיד אִישׁ נֶחְמָד / וְדֶרֶךְ תְּשׁוּבָה לָמַד / וַיְשַׁמֵּשׁ אֶת</p></blockquote>
<blockquote><p>גְּדוֹל הַחָכְמָה / וַיְהִי לָהֶם לְחוֹמָה. / וְהַמַּשִּׂיא בִתּוֹ לְעַם</p></blockquote>
<blockquote><p>הָאָרֶץ כְּאִלּוּ כוֹפְתָהּ בִּפְנֵי הָאֲרִי / כָּעֵת בַּמָּרוֹם</p></blockquote>
<blockquote><p>תַּמְרִיא. / וּכְבָר אָמְרוּ, / נִמְנוּ וְגָמְרוּ: / אֵין בּוּר יְרֵא</p></blockquote>
<blockquote><p>חֵטְא וְלֹא עַם הָאָרֶץ חָסִיד. / וְאָמְרוּ: אִם תַּלְמִיד חָכָם נָחָשׁ</p></blockquote>
<blockquote><p>הוּא – חָגְרֵהוּ עַל מָתְנֶיךָ, / וְאִם עַם הָאָרֶץ חָסִיד – אַל תָּדוּר</p></blockquote>
<blockquote><p>בִּשְׁכוּנָתוֹ וְהַרְחֵק מִמֶּנּוּ מְעוֹנֶךָ. / וְעַל זֶה צָרִיךְ</p></blockquote>
<blockquote><p>לְהַשִּׂיא בַּת הַחֲמוּדָה / לְמַשְׂכִּיל עַל דָּבָר נוֹצֵר תְּעוּדָה /</p></blockquote>
<blockquote><p>״אֲשֶׁר יַכֶּה קִרְיַת סֵפֶר וּלְכָדָהּ״. / וַיַּעַשׂ הֶחָבֵר חֻפָּה</p></blockquote>
<blockquote><p>רַעֲנָנָה וּשְׁלֵמָה / עַד מְלֹאות לוֹ שָׁנָה תְּמִימָה, / וְכָל</p></blockquote>
<blockquote><p>הַתַּלְמִידִים גָמְלוּ אוֹתוֹ חֶסֶד לְרֹב אֵהֲבָתוֹ / וַיֹּאכְלוּ עִמּוֹ</p></blockquote>
<blockquote><p>לֶחֶם בְּבֵיתוֹ.</p> <br/></blockquote>
<blockquote><p>וַיְהִי אַחֲרֵי הַדְּבָרִים הָאֵלֶּה וַיִּשְׁמְעוּ בְנֵי</p></blockquote>
<blockquote><p>הַמְּדִינָה / מַהֲלַל הַכְּנֶסֶת וְהָעֲדִינָה / וְהַמִּדְרָשׁ הַקָּבוּעַ</p></blockquote>
<blockquote><p>וְהַמְּסִבָּה / אֲשֶׁר הַנָּבִיא הַזָּקֵן יוֹשֵׁב בְּקִרְבָּהּ – /</p></blockquote>
<blockquote><p>וַיִּתְקַבְּצוּ כֻלָּם בְנֶפֶשׁ נְכוֹנָה וְרוּחַ נְדִיבָה / אֶל הַנָּהָר</p></blockquote>
<blockquote><p>הַבָּא אֶל הָאַהֲבָה. / וַיִּשְׁלְחוּ מַלְאָכִים אֶל הַנָּזִיר</p></blockquote>
<blockquote><p>וְהֶחָבֵר, / מוֹר עוֹבֵר, /וַיֹּאמְרוּ ״רָאֹה רָאִינוּ כִּי הָיָה יְיָ</p></blockquote>
<blockquote><p>עִמָּכֶם, / וַנֹּאמַר: תְּהִי נָא אָלָה בֵּינוֹתֵינוּ וּבֵינֵיכֶם. /</p></blockquote>
<blockquote><p>בּוֹאוּ וּרְשׁוּ הָאָרֶץ הָעֲרוּכָה / וּשְׁבוּ בְתוֹכָהּ, / וְכָל אִישׁ</p></blockquote>
<blockquote><p>אֲשֶׁר פִּיכֶם יַמְרֶה – / יָרֹה יִיָּרֶה, / וְתִהְיוּ בֵינֵינוּ</p></blockquote>
<blockquote><p>לְמוֹפֵת וּלְאוֹת / שָׂרֵי אֲלָפִים וְשָׂרֵי מֵאוֹת״.</p> <p>וַיֹּאמֶר הַנָּזִיר ״הִנֵּה נָא זָקַנְתִּי, / יוֹם</p></blockquote>
<blockquote><p>מוֹתִי לֹא יָדַעְתִּי, / וְאֵינִי יָכוֹל לְהַלְאוֹת עַצְמִי / וְלָנוּד</p></blockquote>
<blockquote><p>מִמְּקוֹמִי: / אָמְנָם חֲתָנִי הֶחָבֵר הַנֶּאֱמָן / הִנֵּה מוּכָן</p></blockquote>
<blockquote><p>וּמְזֻמָּן, / יָכוֹל לָלֶכֶת בַּהֲנָחָה / לָתוּר לָכֶם מְנוּחָה״. /</p></blockquote>
<blockquote><p>וַיֹּאמֶר הֶחָבֵר ״אֵיתָן מוֹשָׁבֶךָ, / חַי יְיָ וְחֵי נַפְשְׁךָ אִם</p></blockquote>
<blockquote><p>אֶעֶזְבֶךָּ!״ / וַיֹּאמְרוּ לוֹ הָאֲנָשִׁים ״אַל נָא תִּמָּנַע מֵהֲלֹךְ</p></blockquote>
<blockquote><p>אִתָּנוּ, / כִּי אֱלֹהִים הֲדִמָּנוּ, / וְלֹא נִשְׁאַר לָנוּ חָכָם</p></blockquote>
<blockquote><p>מוֹרֶה / וְלֹא תַלְמִיד קוֹרֵא, / וְאַתָּה תִהְיֶה לָּנוּ לְמֵלִיץ</p></blockquote>
<blockquote><p>וּלְסַבָּר – / עַל כֵּן יָדַעְתָּ חֲנוֹתֵנוּ בַּמִּדְבָּר״. / וַיֹּאמֶר</p></blockquote>
<blockquote><p>הַנָּזִיר ״הִנֵּה נָא הָעִיר הַזֹּאת קְרוֹבָה, / וְעָלֶיךָ הַמִּצְוָה</p></blockquote>
<blockquote><p>וְהַחוֹבָה / לְהַדְרִיכָם בְּדֶרֶךְ טוֹב וְנַעֲלָה / וְאֹרַח יְשָׁרִים</p></blockquote>
<blockquote><p>סְלוּלָה, / וְכָל זְמָן תּוּכַל לַעֲלוֹת אֶל בֵּית מְלוֹנַי, / כִּי</p></blockquote>
<blockquote><p>תָבוֹא לִרְאוֹת פָּנַי״.</p> <p>וַיָקָם הֶחָבֵר וַיֵּלֶךְ בַּחֲבוּרָתָם / בִּמְחוֹקֵק</p></blockquote>
<blockquote><p>בְּמִשְׁעֲנוֹתָם. / וַיָּכִינוּ לוֹ בַּיִת וַעֲבָדִים מְשֻׁבָּחִים /</p></blockquote>
<blockquote><p>וַעֲלִיּוֹת מְרֻוָּחִים. / וַיְהִי לְכָל בְּנֵי הָעִיר מָגֵן וְצִנָּה, /</p></blockquote>
<blockquote><p>נָחָה שָׁקְטָה הָאָרֶץ פָּצְחוּ רִנָּה, / וַיִּנְהָגֵם עַל פִּי</p></blockquote>
<blockquote><p>הַתּוֹרָה וְהַמּוּסָר / וַיְהִי לָהֶם לְשַׂר, / וַיּוֹרֵם אֶת מִשְׁפַּט</p></blockquote>
<blockquote><p>הַתְּשׁוּבָה / כַּאֲשֶׁר הִיא עַל סֵפֶר הַיָּשָׁר כְּתוּבָה. /</p></blockquote>
<blockquote><p>וְלִפְעָמִים הָיָה מַקְבִּיל פְנֵי חֲבֵרוֹ בְּנַחַת וְזִמְרָה / חֹדֶשׁ</p></blockquote>
<blockquote><p>וְשַׁבָּת קְרֹא מִקְרָא. / וַיִּהְיוּ לוֹ בָּנִים וְתַלְמִידִים /</p></blockquote>
<blockquote><p>עוֹסְקִים בַּתּוֹרָה וּבִגְמִילוּת חֲסָדִים. / וַיְהִי רוֹכֵב פַּעַם</p></blockquote>
<blockquote><p>בַּשָּׁבוּעַ, / בִּזְמָן קָבוּעַ, / לְיַשֵּׁר אֶת בְּנֵי הַסְּחוֹרָה /</p></blockquote>
<blockquote><p>בַּמִּדָּה בַּמִּשְׁקָל וּבַמְּשׂוּרָה / וְשָׁב אֶל מִדְרָשׁוֹ שׁוֹקֵט</p></blockquote>
<blockquote><p>וְשָׂמֵחַ / וְרֹב שָׁלוֹם עַד בְּלִי יָרֵחַ.</p> <br/></blockquote>
<p>מקור: שירמן – <b>השירה העברית בספרד ובפרובאנס</b></p>
<h2>שער שלישי</h2>
<h3>אָמְרוּ כִּי הָיָה אִישׁ אֶחָד בְּאֶרֶץ תּוֹגַרְמָה (מתוך השער השלישי)</h3>
<blockquote><p>אָמְרוּ כִּי הָיָה אִישׁ אֶחָד בְּאֶרֶץ תּוֹגַרְמָה /</p></blockquote>
<blockquote><p>הָיָה עוֹבֵד אֲדָמָה, / וַיְהִי שׁוֹכֵן בִּכְפָרִים / וּבָתֵּי</p></blockquote>
<blockquote><p>הַחֲצֵרִים, / וְהָיָה מִתְפַּתֶּה וּמִתְהוֹלֵל בַּעֲמָלוֹ / אָדָם אֲשֶׁר</p></blockquote>
<blockquote><p>לֹא יָדַע בֵּין יְמִינוֹ לִשְׂמֹאלוֹ, / וּבֵינוֹ וּבֵין עִיר</p></blockquote>
<blockquote><p>הַסּוֹחֲרִים הָרָמִים / מַהֲלָךְ שְׁלֹשֶׁת יָמִים. / וְשָׁנָה אַחַת</p></blockquote>
<blockquote><p>גִּדַּל בְּבֵיתוֹ זְרָעִים / לַעֲבוֹדַת הַמֶּשִׁי יְדוּעִים / וַיָּרֻם</p></blockquote>
<blockquote><p>תּוֹלָעִים, / וַתְּהִי לְחֶפְצוֹ הָעֲבוֹדָה נִמְשָׁכָה / וַתִּשְׁלַם</p></blockquote>
<blockquote><p>כָּל הַמְּלָאכָה.</p> <p>וַיָּקָם יוֹם אֶחָד לָלֶכֶת בְּחָפְזָה / אֶל הָעִיר</p></blockquote>
<blockquote><p>הָעַלִּיזָה / לִמְכֹּר הַמֶּשִׁי הַגָּדֵל בְּהֵיכָלוֹ / לְהַרְבּוֹת</p></blockquote>
<blockquote><p>חֵילוֹ. / וַיְמַהֵר לְהָפִיק רְצוֹנוֹ, / וַיַּחֲבֹשׁ אֶת אֲתוֹנוֹ, /</p></blockquote>
<blockquote><p>וַיָּשִׂים אֶת הַמֶּשִׁי בָּאַמְתַּחַת / בְּשׁוּבָה וָנַחַת, / וַיִּתֵּן</p></blockquote>
<blockquote><p>אֲבָנִים בַּצַּד הָאַחֵר / וּלְמִשְׁקַל הַמֶּשִׁי אוֹתָם בּוֹחֵר, /</p></blockquote>
<blockquote><p>וַיִּקַּח צֵידָה לִמְזוֹנוֹ / וַיַעֲמֹס בַּשָּׁוֶה עַל אֲתוֹנוֹ. /</p></blockquote>
<blockquote><p>וַיְצַו לְבֵיתוֹ וַיֵּלֶךְ / דֶּרֶךְ הַמֶּלֶךְ. / וַיְהִי הוּא הוֹלֵךְ</p></blockquote>
<blockquote><p>לְבַדּוֹ – / וְהִנֵּה אִישׁ אֶחָד קָרֵב לְנֶגְדּוֹ, / וְהוּא עָנִי</p></blockquote>
<blockquote><p>מִסְכֵּן, / בְּנַפְשׁוֹ מִסְתַּכֵּן. / וַיִּירָא מְאֹד הָעוֹבֵד / כִּי</p></blockquote>
<blockquote><p>רָאָה אוֹתוֹ מִבְּלִי לְבוּשׁ אוֹבֵד / וַיֹּאמֶר אֶל הָאִישׁ ״הֲשָׁלוֹם?</p></blockquote>
<blockquote><p>/ גֶּשׁ נָא הֲלֹם!״ / וַיֹּאמֶר לוֹ הָאִישׁ ״אַל תִּירָא מִפָּנַי / כִּי</p></blockquote>
<blockquote><p>שָׁלוֹם לְךָ וְאֵין דָּבָר, חַי יְיָ!״</p> <br/></blockquote>
<blockquote><p>וַיֹּאמֶר הָעוֹבֵד ״לְאֵיזֶה מָקוֹם מְגַמָּתֶךָ? /</p></blockquote>
<blockquote><p>אוּלַי אוּכַל לָלֶכֶת בְּחֶבְרָתֶךָ״. / וַיֹּאמֶר ״אֶל הַסְּחוֹרָה</p></blockquote>
<blockquote><p>מְגַמָּתִי / וּלְפִי מִשְׁקַל הַמֶּשִׁי תַּחֲנוֹתִי״. / וַיִּשְׂמַח</p></blockquote>
<blockquote><p>הָעוֹבֵד עַל הַבְּשׂוֹרָה / וְאָרַח עִמּוֹ לְחֶבְרָה. / וַיֹּאמֶר לוֹ</p></blockquote>
<blockquote><p>״הִנְנִי בְחֶבְרָתְךָ בוֹטֵחַ / וּבְאַהֲבָתְךָ שָׂמֵחַ. / וְעַתָּה</p></blockquote>
<blockquote><p>הוֹדִיעֵנִי מְלַאכְתְּךָ וְעִנְיָנֶיךָ, / אִם נָא מָצָאתִי חֵן</p></blockquote>
<blockquote><p>בְּעֵינֶיךָ״. / וַיֹּאמֶר הָאִישׁ ״מִנְּעוּרַי אֲנִי שׁוֹכֵן רָמִים /</p></blockquote>
<blockquote><p>עִיר גְּדוֹלָה שֶׁל חֲכָמִים / וְשָׁם הֲכִינוֹתִי מְנוּחָתִי</p></blockquote>
<blockquote><p>וְשִׁבְתִּי, / וַאֲנִי סוֹפֵר מָהִיר בִּמְלַאכְתִּי, / וְיֶשׁ לִי</p></blockquote>
<blockquote><p>בְתוֹגַרְמָה אָח וְקָרוֹב / וְרֵעִים לָרֹב: / וְהִנְנִי שׁוֹחֵר</p></blockquote>
<blockquote><p>לִקְרָאתָם / לִדְרֹשׁ שְׁלוֹמָם וְטוֹבָתָם״. / וַיֹּאמֶר ״הוֹדִיעֵנִי</p></blockquote>
<blockquote><p>דָבָר בִּמְלַאכְתְּךָ הָרָמָה / וְיַעֲבֹר עָלַי מָה…</p> <p>וַיֹּאמֶר הָעוֹבֵד ״רְאֵה דְבָרֶיךָ טוֹבִים וּנְכוֹחִים</p></blockquote>
<blockquote><p>בְּמִלּוּלָךְ / וְשׁוֹמֵעַ אֵין לָךְ! / כִּי הֶעָמָל הִפְלִיאַנִי</p></blockquote>
<blockquote><p>הַפְלֵא וָפֶלֶא / וְלֹא נִסִּיתִי לָלֶכֶת בָּאֵלֶּה, / וְאֵינִי מַכִּיר</p></blockquote>
<blockquote><p>בְּדִבּוּרְךָ לְבַד חִתּוּךְ שְׂפָתַיִם / פַּעַם וּשְׁתַיִם״. / וַיְהִי</p></blockquote>
<blockquote><p>הֵם הוֹלְכִים הָלוֹךְ וְדַבֵּר / וְכָל אֶחָד בַּחֲבֵרוֹ מִתְחַבֵּר – /</p></blockquote>
<blockquote><p>וַתִּצְנַח הָאָתוֹן וַתִּכְרַע / וַיֶּחֱלַשׁ כֹּחָהּ וַיִּגָּרַע, / כִּי</p></blockquote>
<blockquote><p>לֹא יָכְלָה לִסְבֹּל הַמַּשָּׂא. / וְעֵין הָעוֹבֵד עָלֶיהָ חָסָה /</p></blockquote>
<blockquote><p>וַיִּצְעַק וַיֶּהֱמֶה לְקוֹלָהּ / וַיָשֶׁת לִבּוֹ לְחִילָהּ.</p> <br/></blockquote>
<blockquote><p>וַיֹּאמֶר הַסּוֹפֵר ״לָמָּה תִּשּׁוֹמֵם, / חָרֵד</p></blockquote>
<blockquote><p>וְהוֹמֵם? / קְרַב אֶל הָאָתוֹן וְהָסֵר הַמַּשָּׂא אֲשֶׁר עָלֶיהָ – /</p></blockquote>
<blockquote><p>וְתָנוּחַ מְעַט מִזְעָר מֵחֲבָלֶיהָ״. / וַיִּגַּשׁ הָעוֹבֵד לְהָסִיר</p></blockquote>
<blockquote><p>הַמַּשָּׂא מֵעַל אֲתוֹנוֹ / וְהַסּוֹפֵר עֲזָרוֹ כְּפִי כֹחוֹ וְאוֹנוֹ. /</p></blockquote>
<blockquote><p>וְכִרְאוֹת בְּאַמְתַּחְתּוֹ אֶבֶן מַעֲמָסָה / וַיֹּאמֶר לוֹ ״מָה</p></blockquote>
<blockquote><p>הַמַּשָּׂא?״ / וַיֹּאמַר ״אַמְתַּחְתִּי הָאֶחָד מָלֵא מֵהַמֶּשִׁי הַקַּל</p></blockquote>
<blockquote><p>– / וְהָאֲבָנִים לְנֶגְדּוֹ לְמִדָּה וּלְמִשְׁקָל״. / וַיֹּאמֶר</p></blockquote>
<blockquote><p>הַסּוֹפֵר ״לָמָּה מַחְשְׁבוֹתֶיךָ חֲפֵצוֹת / בְּאַבְנֵי גִּיר</p></blockquote>
<blockquote><p>מְנֻפָּצוֹת?״ / וַיֹּאמֶר הָעוֹבֵד ״וְאֵיךְ יִנָּשֵׂא הַמֶּשִׁי לְבַדּוֹ</p></blockquote>
<blockquote><p>/ וְאֵין מִשְׁקָל לְנֶגְדּוֹ?״ / וַיֹּאמַר ״בְּעִנְיַן הַמַּשָּׂא</p></blockquote>
<blockquote><p>אֲלַמֶּדְךָ דְּבָרִים / שְׁמַע בְּקוֹלִי אִיעָצְךָ מֵישָׁרִים״. /</p></blockquote>
<blockquote><p>וַיְצַו לְהָרִים הָאֲבָנִים מִן הָאַמְתַּחַת / וּלְהַצִּיל נֶפֶשׁ</p></blockquote>
<blockquote><p>בְּהֶמְתּוֹ מִנִּי שַׁחַת / וּלְחַלֵּק הַמֶּשִׁי בְּאַמְתְּחוֹתָיו /</p></blockquote>
<blockquote><p>מִשְּׁתֵי רוּחוֹתָיו. / וַיַּעַשׂ הָעוֹבֵד כְּפִי מַאֲמָרוֹ וּרְצוֹנוֹ /</p></blockquote>
<blockquote><p>וַיֵּקַל הַמַּשָּׂא מֵעַל אֲתוֹנוֹ.</p> <p>וַיָּקוּמוּ וַיֵּלְכוּ שְׁנֵיהֶם חֲבֵרִים / וְהָאָתוֹן</p></blockquote>
<blockquote><p>לִפְנֵיהֶם מַעֲבִירִים. / וְהַסּוֹפֵר הִשְׁלִים חִידָתוֹ הַנִּפְלָאָה /</p></blockquote>
<blockquote><p>לְהַעֲבִיר מֵהֶם יָגוֹן וּתְלָאָה, / וְהָעוֹבֵד תָּמַהּ מְאֹד</p></blockquote>
<blockquote><p>מֵחִדּוּדוֹ / וְגֹדֶל תְּבוּנָתוֹ וְהוֹדוֹ / וַיֹּאמַר ״אַחֲרֵי אֲשֶׁר</p></blockquote>
<blockquote><p>הוֹד תְּבוּנַת לְשׁוֹנְךָ גָדַל / מַדּוּעַ אַתָּה כָּכָה דַּל?״ /</p></blockquote>
<blockquote><p>וַיֹּאמֶר הַסּוֹפֵר ״וַהֲלֹא חֲכָמִים עָשׂוּ תַעֲנִיּוֹת וּצְדָקוֹת /</p></blockquote>
<blockquote><p>עַל הַסּוֹפְרִים וְעַל מְלַמְּדֵי תִינוֹקוֹת / שֶׁלֹּא יִתְעַשְּׁרוּ</p></blockquote>
<blockquote><p>בִמְלַאכְתָּם, / וְכֵן נִכְתַּב וְנֶחְתַּם! / וַעֲלֵיהֶם אָמְרוּ: נָאָה</p></blockquote>
<blockquote><p>עֲנִיּוּת לְיִשְׂרָאֵל וְצַעֲרָא / כְּוַרְדָא סוּמְקָא לְסוּסְיָא</p></blockquote>
<blockquote><p>חִוְרָא״. / וַיֹּאמֶר הָעוֹבֵד ״הוֹאִיל וְזֹאת הִיא מַחֲלָתֶךָ – /</p></blockquote>
<blockquote><p>הָעֲבוֹדָה! לֹא אֲקַבֵּל עֲצָתֶךָ / וְלֹא אֶתְחַבֵּר עוֹד לְחֶבְרָתֶךָ.</p></blockquote>
<blockquote><p>/ וְעַתָּה הִפָּרֵד נָא מֵעָלַי / אַל תּוֹסֶף עוֹד דַּבֵּר אֵלַי /</p></blockquote>
<blockquote><p>וְלֵךְ לְשָׁלוֹם דֶּרֶךְ מְגַמָּתְךָ עֲרוּכָה – / וַאֲנִי אֶתְנַהֲלָה</p></blockquote>
<blockquote><p>לְאִטִּי לְרֶגֶל הַמְּלָאכָה״. / וַיֵּלֶךְ הָאִישׁ לְבַדּוֹ אֹרַח</p></blockquote>
<blockquote><p>נְתִיבוֹ וַיִּתְפָּרְדוּ אִישׁ מֵעַל אוֹהֲבוֹ.</p> <p>וַיָּשָׁב הָעוֹבֵד וַיִּקַּח אֶת הָאֲבָנִים אֲשֶׁר</p></blockquote>
<blockquote><p>הִנִּיחַ / וְנֶגֶד עֲצַת הַסּוֹפֵר דְּבָרִים הִטִּיחַ / לְלַקֵּט אוֹתָם</p></blockquote>
<blockquote><p>בְּמִסְפָּר / מַעֲרֵמַת הֶעָפָר / וַיִּתֵּן אוֹתָם בְּאַמְתַּחְתּוֹ /</p></blockquote>
<blockquote><p>אֲשֶׁר נִמְצָא אִתּוֹ / וּבְמִשְׁקַל הַמֶּשִׁי אוֹתָם סִלָּה / כַּאֲשֶׁר</p></blockquote>
<blockquote><p>הָיָה בַתְּחִלָּה. / וַיָּקָם וַיֵּלֶךְ לְבַדּוֹ מְעַט מְעַט / לְרֶגֶל</p></blockquote>
<blockquote><p>הָאָתוֹן לְאַט, / וַיַּעַשׂ בַּדֶּרֶךְ שְׁלֹשָׁה יָמִים / וּבָרָק</p></blockquote>
<blockquote><p>מִמְּרוֹרָתוֹ יַהֲלֹךְ עָלָיו אֵימִים. / וַיַּגִּיעַ אֶל הָעִיר יָגֵעַ,</p></blockquote>
<blockquote><p>/ עֹנִי וְעָמָל שָׂבֵעַ. / וַיֵּלֶךְ בַּלַּיְלָה הַהוּא בִמְלוֹנוֹ /</p></blockquote>
<blockquote><p>וַיַּאֲכֵל לַאֲתוֹנוֹ.</p> <p>וַיְהִי בַבֹּקֶר וַיָּבוֹא הַסּוֹחֵר, וַיִּקְנֶה</p></blockquote>
<blockquote><p>הַמֶּשִׁי מִיָּדוֹ / וַיִּשְׁקֹל הַכֶּסֶף בַּעֲדוֹ / וַיֹּאמַר ״יֵשׁ</p></blockquote>
<blockquote><p>עוֹד בְּאַמְתַּחְתְּךָ דָּבָר לְמָכְרוֹ? / וַאֲנִי אֶתֵּן שְׂכָרוֹ /</p></blockquote>
<blockquote><p>וְאַרְבֶּה כֶּסֶף שִׁבְרוֹ״. / וַיֹּאמַר ״אֵין בּוֹ לְבַד דָּבָר נָקַל –</p></blockquote>
<blockquote><p>/ אַבְנֵי מִשְׁקָל / הֵבֵאתִי לְיַשֵּׁר הָמַּשָּׂא, / אֶבֶן מַעֲמָסָה״.</p></blockquote>
<blockquote><p>/ וַיִּגַּשׁ הַסּוֹחֵר לִרְאוֹתָם / וְהִנֵּה הֵם שׁוֹנוֹת בְּמַרְאִיתָם!</p></blockquote>
<blockquote><p>/ וַיָּבֶן כִּי הֵם טוֹבוֹת / לְחַדֵּד וְלִמְרֹט הַחֲרָבוֹת. /</p></blockquote>
<blockquote><p>וְהַמֶּלֶךְ צִוָּה עַל כָּל בְּנֵי הָעִיר / יַחַד יָשִׁישׁ וְצָעִיר /</p></blockquote>
<blockquote><p>לְתַקֵּן כָּל כְּלֵי מִלְחָמָה וּלְחַדְּדָם / לְהַרְאוֹת הֲדָרָם</p></blockquote>
<blockquote><p>וְהוֹדָם / וּלְתַקֵּן כָּל נֶפֶשׁ אֲשֶׁר לְבוּשׁ שִׁרְיוֹן עוֹטָה /</p></blockquote>
<blockquote><p>חֶרֶב חֲדוּדַָה וְגַם מֹרָטָּה, / וְכָל הָאֲנָשִׁים אֲשֶׁר לֹא יַרְאוּהָ</p></blockquote>
<blockquote><p>בִזְמָן קָבוּעַ / חֶרְפָּתוֹ וְעָנְשׁוֹ יָדוּעַ, / כִּי כָל חֹשֶׁךְ</p></blockquote>
<blockquote><p>לִצְפוּנָיו טָמוּן / וּבָתֵיהֶם יִתְשָׂמוּן. / וּבִרְאוֹת הָאִישׁ</p></blockquote>
<blockquote><p>הָאֲבָנִים הַסְּפוּרוֹת / אֲשֶׁר הָיוּ מִזְּמָן רַב נֶעְדָּרוֹת /</p></blockquote>
<blockquote><p>וּבְעֵת הַצֹּרֶךְ הֵמָּה נִסְתָּרוֹת, / אֲבָנִים גְּדוֹלוֹת אֲבָנִים</p></blockquote>
<blockquote><p>יְקָרוֹת ־ ־ / דִּבֶּר אֶל הָעוֹבֵד דְּבָרִים טוֹבִים / אֶל לֵב כָּל</p></blockquote>
<blockquote><p>מַשְׂכִּיל קְרוֹבִים / וַיֹּאמַר ״נָקְבָה אֵלַי בְּאֵלֶּה הָאֲבָנִים</p></blockquote>
<blockquote><p>מְחִירֶךָ / וַאֲנִי אֶתֵּן אֶת שְׂכָרֶךָ״. / וַיְהִי בְעֵינֵי הָעוֹבֵד</p></blockquote>
<blockquote><p>לִשְׂחוֹק / וַיָּבִיא עֵצוֹת מֵרָחוֹק / וַיֹּאמַר ״אֲנִי אֶמְכְּרֵם</p></blockquote>
<blockquote><p>בַּעֲבוּר נַעֲלַיִם / וּבִמְחִיר הַמֶּשִׁי כִּפְלַיִם״. / וַיֹּאמֶר</p></blockquote>
<blockquote><p>הַסּוֹחֵר ״אֲשַׁלֵּם לְךָ כָּל אֲשֶׁר תִּשְׁאַל / וְאַל תָּשִׂים</p></blockquote>
<blockquote><p>מִלָּתְךָ לְאַל״. / וַיִּקַּח הָאֲבָנִים וַיּוֹצִיאֵם לַחוּץ, / כִּי</p></blockquote>
<blockquote><p>הָיָה דְבַר הַמֶּלֶךְ נָחוּץ, / וַיִּתֵּן לָעוֹבֵד כְּסוּתוֹ וּמִנְעָלוֹ</p></blockquote>
<blockquote><p>/ וְכֶסֶף הִרְבָּה לוֹ. / וַיִּשְׂמַח הָעוֹבֵד בְּסִכְלוֹ וַיִּתְהַלָּל</p></blockquote>
<blockquote><p>/ כִּי הוֹלִיךְ יוֹעֲצִים שׁוֹלָל / וְסוֹפְרִים יְהוֹלָל / וַיֹּאמַר</p></blockquote>
<blockquote><p>״זֶה תַגְמוּל כָּל עָמֵל וְכָל רוֹעֵץ / אֲשֶׁר לֹא שָׁמַע לְקוֹל</p></blockquote>
<blockquote><p>יוֹעֵץ״. / וַיָּשָׁב לִבּוֹ שָׂמֵחַ / וּבְהוֹלְלוּתוֹ מִשְׁתַּבֵּחַ /</p></blockquote>
<blockquote><p>וַיְהִי לוֹ מִקְנֵה צֹאן וּמִקְנֵה בָקָר / שִׂמְחָה וְשָׂשׂוֹן</p></blockquote>
<blockquote><p>וִיקָר.</p> <br/></blockquote>
<h3>אִישׁ הָיָה בִכְפַר חָנָן (מתוך השער השלישי - הכילי שבא על ענשו)</h3>
<blockquote><p>אִישׁ הָיָה בִכְפַר חָנָן / בְּצֵל הַמָּמוֹן יִתְלוֹנָן,</p></blockquote>
<blockquote><p>/ וַיְהִי לוֹ בֵית חֶמְדָּה / וּמִקְנֵה בָקָר וַעֲבֻדָּה. / וַיְהִי</p></blockquote>
<blockquote><p>כִילַי בִּמְמוֹנוֹ וּמְאֹדוֹ, / מַשְׁחִית תִּפְאַרְתּוֹ וְהוֹדוֹ, /</p></blockquote>
<blockquote><p>וְדַעְתּוֹ בֶּאֱמוּנָתוֹ נִבְדֶּלֶת / בְּהַרְבּוֹת דְּבָרִים לְאֵין</p></blockquote>
<blockquote><p>תּוֹעֶלֶת, / וּמְמוֹן עָנִי וְאֶבְיוֹן יְפַזֵּר / וְיָתוֹם וְאֵין עוֹזֵר</p></blockquote>
<blockquote><p>/ בְרֹב דְּבָרָיו הָיָה עָצֵל וְכָבֵד / כִּילַי אוֹבֵד.</p> <p>וַיְהִי הַיּוֹם וַיָּבוֹא אֶל בֵּיתוֹ הֵלֶךְ / כְּתֹאַר</p></blockquote>
<blockquote><p>בְּנֵי הַמֶּלֶךְ / אִישׁ אֶחָד נִכְבָּד מִבְּנֵי עַמּוֹ, / וּשְׁנֵי</p></blockquote>
<blockquote><p>נְעָרָיו עִמּוֹ. / וַיֹּאמֶר לוֹ ״הַאֶמְצָא מָלוֹן הַלַּיְלָה הַזֶּה</p></blockquote>
<blockquote><p>בְקִרְבֶּךָ?״ וַיֹּאמַר ״הוֹאֶל נָא וְלִין וְיִיטַב לִבֶּךָ!״</p></blockquote>
<blockquote><p>וְהָאוֹרֵחַ לֹא הֵבִין סוֹד תַּחְבּוּלוֹתָיו / שֶׁיּוֹרֵד עַל פִּי</p></blockquote>
<blockquote><p>מִדּוֹתָיו. / וַיִּתֵּן לְתַחֲנוֹתָיו מִטָּה מֻצַּעַת וְיָפָה, /</p></blockquote>
<blockquote><p>וַיַּעַשׂ לוֹ מִשְׁתֶּה וּמַצּוֹת אָפָה. / וַיּוֹסֶף לְכַבְּדוֹ</p></blockquote>
<blockquote><p>וּלְהַדְּרוֹ / וַיִּתֵּן מִסְפּוֹא לַחֲמוֹרוֹ, / וַיִּתֵּן מַיִם לִרְחֹץ</p></blockquote>
<blockquote><p>רַגְלָיו / וְרַגְלֵי הָאֲנָשִׁים הַנִּלְוִים אֵלָיו, / וַיֹּאכְלוּ</p></blockquote>
<blockquote><p>כֻלָּם בְּבֵיתוֹ וְאוּלַמּוֹ / וַיִּשְׁתּוּ וַיִּשְׁכְּרוּ</p></blockquote>
<blockquote><p>עִמּוֹ.</p> <p>וַיְהִי אַחֲרֵי הַסְּעוּדָה / וַיִּפְתַח הָאוֹרֵחַ</p></blockquote>
<blockquote><p>בְּמִשְׁלֵי תְעוּדָה / וַיֹּאמֶר אֶל הָאִישׁ ״שְׁמַע נָא דְבָרַי, /</p></blockquote>
<blockquote><p>יֹשֶׁר לִבִּי אֲמָרַי: / הִנֵּה אַתָּה בַהֲדָרְךָ כִּבָּדְתָּנוּ /</p></blockquote>
<blockquote><p>וּמִטּוּב נָעָמְךָ הֶאֱכַלְתָּנוּ / וּמִיֵּין רֶקַח נְדִיבוּתְךָ</p></blockquote>
<blockquote><p>הִשְׂבַּעְתָּנוּ / וּמִמֵּיטַב פְּעֻלָּתְךָ שִׂמַּחְתָּנוּ. / עַל כֵּן</p></blockquote>
<blockquote><p>רָאוּי לְהוֹדִיעֲךָ דְבָרִים עֲרֵבִים, / גּוֹמֵל חֲסָדִים טוֹבִים, /</p></blockquote>
<blockquote><p>לְמַעַן תָּבִין בְּמַהֲלַל לִבִּי וְדַרְכּוֹ / לְמִי עוֹלַלְתָּ כֹּה״. /</p></blockquote>
<blockquote><p>וַיֹּאמַר ״דַּבֵּר כְּפִי צִדְקְךָ וְחַסְדֶּךָ, / כִּי שׁוֹמֵעַ</p></blockquote>
<blockquote><p>עַבְדֶּךָ״.</p> <p>וַיֹּאמֶר הָאוֹרֵחַ ״אָמְרוּ הַקַּדְמוֹנִים / אֲשֶׁר</p></blockquote>
<blockquote><p>בְּמַחֲנֵה הַחָכְמָה חוֹנִים / טוּב הַמָּלוֹן וּשְׁלֵמוּתוֹ / קוֹנֶה</p></blockquote>
<blockquote><p>בֶן־חֹרִין בִּנְדִיבוּתוֹ, / וּפְרִי הַנְּדִיבוּת וְהָרֶוַח / הוּא</p></blockquote>
<blockquote><p>הַמַּהֲלָל וְהַשֶּׁבַח. / וְאָמְרוּ: נְדִיבוּת הַנִּכְבָּדִים – /</p></blockquote>
<blockquote><p>גְּמִילוּת חֲסָדִים, / וּמִפְעָלוֹת הַנֶּאֱהָבוֹת – / גֶּשֶׁם נְדָבוֹת,</p></blockquote>
<blockquote><p>/ כִּי הַזּוֹרֵעַ נְדָבָה / יִקְצֹר הוֹד וְאַהֲבָה. / וְאָמְרוּ:</p></blockquote>
<blockquote><p>הַנָּדִיב יְבַקֵּשׁ הַחֵן וְהַיֹּשֶׁר / וְהַכִּילַי – הַמָּמוֹן</p></blockquote>
<blockquote><p>וְהָעֹשֶׁר. / וְהַמִּתְנַדֵּב בְּנַפְשׁוֹ יְהֻלַּל / וְקוֹבֵץ עַל יָד –</p></blockquote>
<blockquote><p>חֶלְקוֹ יְקֻלַּל. / וְאָמְרוּ: הַנְּדִיבוּת יְחַזֵּק הַבֶּדֶק /</p></blockquote>
<blockquote><p>וּמִשָּׁם נוֹלַד הַצֶּדֶק. / הַצֶּדֶק יַצְמִיחַ הַשֶּׁבַח וְהַכֹּשֶׁר /</p></blockquote>
<blockquote><p>וּמִמֶּנּוּ יִוָּלֵד הַיֹּשֶׁר. / הַיֹּשֶׁר יוֹלִיד הַנֵּצַח וְהָעֶזְרָה</p></blockquote>
<blockquote><p>/ וּמִמֶּנּוּ תֵצֵא הָחֲבוּרָה. / הַחֲבוּרָה – גֶּפֶן חֲזָקָה /</p></blockquote>
<blockquote><p>תַּצְמִיחַ צְדָקָה. / הַצְּדָקָה תָּבוֹא הֲלֹם / וְתוֹסִיף אַהֲבָה</p></blockquote>
<blockquote><p>וְשָׁלוֹם. / הַשָּׁלוֹם – סוֹד הַתְּבוּנָה / וְעִקַּר הָאֱמוּנָה. /</p></blockquote>
<blockquote><p>הָאֱמוּנָה מוּכֶנֶת לְהָבִין וּלְהוֹרוֹת / וְהִיא רֹאשׁ הַתּוֹרוֹת. /</p></blockquote>
<blockquote><p>הַתּוֹרוֹת – מִשְׁפָּטִים וְחֻקִּים / מִצּוּף וּמִנֹּפֶת מְתוּקִים, /</p></blockquote>
<blockquote><p>וּבָהֶם יִשּׁוּב הָאֲרָצוֹת / קִרְבַת אֱלֹהִים חֲפֵצוֹת. / רְאֵה כַּמָּה</p></blockquote>
<blockquote><p>הַנְּדִיבוּת נִכְבֶּדֶת / אֲשֶׁר בְּרֹאשׁ הַמַּעֲלוֹת עוֹמֶדֶת /</p></blockquote>
<blockquote><p>וְכַמָּה גְדוֹלָה מַעֲלַת הַשְּׂמֵחִים / בְּהַכְנָסַת אוֹרְחִים!״</p> <br/></blockquote>
<blockquote><p>וַיֹּאמֶר הָאִישׁ אֶל הָאוֹרֵחַ / אֲשֶׁר בִּנְגִינוֹתָיו</p></blockquote>
<blockquote><p>יְנַצֵּחַ / ״הֲיֵּשׁ לְךָ מָחָר לְהַשְׁכִּים, / לְהִתְיַצֵּב בַּבֹּקֶר</p></blockquote>
<blockquote><p>לִבְּךָ מַסְכִּים? / עֵת הוּא לְהָכִין מִטּוֹת וּלְהַצִּיעַ – / אַחַר</p></blockquote>
<blockquote><p>תְּחַוֶּה דַעְתְּךָ וְתַבִּיעַ״. / וַיֹּאמֶר לוֹ ״עוֹד יֶשׁ לִי לְתַקֵּן</p></blockquote>
<blockquote><p>בְּאַמְתַּחְתִּי מִגְרָע / אֲשֶׁר הַיּוֹם נִקְרַע״. / וַיִּקַּח</p></blockquote>
<blockquote><p>אַמְתַּחְתּוֹ לְתָפְרוֹ, / וְשָׁם נִקְרַב הָאִישׁ בַּעַל הַבַּיִת</p></blockquote>
<blockquote><p>לְעָזְרוֹ, / וַיַּרְא בְּתוֹכוֹ צִנְצֶנֶת / לְתַכְלִית הַיֹּפִי</p></blockquote>
<blockquote><p>מְתֻקֶּנֶת. / וַיֹּאמֶר אֶל הָאִישׁ ״מַה זֶּה בְּיָדֶךָ / לְמַעֲלַת</p></blockquote>
<blockquote><p>כְּבוֹדְךָ וְהוֹדֶךָ?״ / וַיֹּאמַר ״אֲנִי רוֹכֵל בְּאֶרֶץ הַחֲוִילָה, /</p></blockquote>
<blockquote><p>נוֹרָא עֲלִילָה. / בַּשָּׁנָה הַזֹּאת חָסֵר בְּעִירִי צְרִי גִלְעָד /</p></blockquote>
<blockquote><p>לְמַרְפֵּא וּלְמִסְעָד, / וַאֲנִי הָלַכְתִּי אֶל גִּלְעָד בַּעֲבוּרוֹ /</p></blockquote>
<blockquote><p>וָאַרְבֶּה זָהָב בִּמְחִירוֹ, / וְזֶהוּ הַצֳּרִי הַגָּדוֹל אֲשֶׁר אֵין</p></blockquote>
<blockquote><p>חֵקֶר לְתִפְאַרְתּוֹ / לְכָל אֲשֶׁר נִמְצָא אִתּוֹ״. / וַיֹּאמֶר לוֹ</p></blockquote>
<blockquote><p>״בְּחַסְדְּךָ הוֹדִיעֵנִי בְתוֹעֲלוֹתָיו דָּבָר נִרְשָׁם / זְעֵיר שָׁם</p></blockquote>
<blockquote><p>זְעֵיר שָׁם״. / וַיֹּאמֶר הָאוֹרֵחַ ״וּמִי יוּכַל לְמַלֵּל גְּבוּרוֹתָיו</p></blockquote>
<blockquote><p>/ וְשֶׁבַח סְגֻלַּת מַעֲלוֹתָיו? / אֲבָל הָרוֹפְאִים נִמְנוּ וְגָמְרוּ</p></blockquote>
<blockquote><p>בְמַהֲלָלָיו אֶחָד וְתִשְׁעִים / לְכָל מַשְׂכִּיל נוֹדָעִים / וְאִם</p></blockquote>
<blockquote><p>תִּרְצֶה לֵדַע סִימָן בְּמַהֲלָלוֹ / ״צֵא״ תֹּאמַר לוֹ / כִּי בְכָל</p></blockquote>
<blockquote><p>מֵעֲלָה וּמַעֲלָה / יוֹעִיל לְכָל כְּאֵב וּלְכָל מַחֲלָה / וְהַשּׁוֹתֶה</p></blockquote>
<blockquote><p>סַם הַמָּוֶת – בְּטוּבוֹ יַצִּילֵהוּ / וּבִשְׁתוֹתוֹ מִמֶּנּוּ מִיָּד</p></blockquote>
<blockquote><p>יוֹעִילֵהוּ״. / וַיֹּאמֶר לוֹ ״תֶּן לִי מְעַט מִזְעָר לְזֵכֶר</p></blockquote>
<blockquote><p>מַעֲלוֹתָיו / לְמַעַן אֲסַפְּרָה כָּל נִפְלְאוֹתָיו״. / וַיֹּאמַר</p></blockquote>
<blockquote><p>״שָׁמַעְתִּי לְקוֹלֶךָ / אֲנִי אֲמַלֵּא אֶת מִשְׁאָלֶךָ״. / וַיִּתֶּן</p></blockquote>
<blockquote><p>לוֹ מִמֶּנּוּ בִּכְלִי קָטֹן / צְלוֹחִית שֶׁל פָּלְיָטוֹן. / וַיֹּאמֶר</p></blockquote>
<blockquote><p>לוֹ ״הֲרֵי לְךָ מֹהַר וּמַתָּן / בִּצְלוֹחִית קְטַנָּה נִתָּן / וְעַתָּה</p></blockquote>
<blockquote><p>שִׂים עֵינֶיךָ לְשָׁמְרֵהוּ / כִּי לֹא דָבָר רֵיק הוּא!״</p> <p>וַיְהִי כְּכַלּוֹת הָאוֹרֵחַ לְתַקֵּן אַמְתַּחְתּוֹ /</p></blockquote>
<blockquote><p>וּבָא וְלָן בְּמִשְׁכַּב תִּפְאַרְתּוֹ – / וַיַּעַל בְּלֵב בַּעַל</p></blockquote>
<blockquote><p>הַבַּיִת / שָׁמִיר וָשַׁיִת. / וַיֹּאמֶר לוֹ ״אֲדוֹנִי, הוֹאִיל וְאֵין</p></blockquote>
<blockquote><p>רְצוֹנְךָ לְהִתְאַחֵר / וּבַבֹּקֶר אַתָּה תְשַׁחֵר – / צַוֵּה לִפְרֹעַ</p></blockquote>
<blockquote><p>הַהוֹצָאָה / אֲשֶׁר לְרַגְלְךָ בָאָה״. / וַיֹּאמַר ״מַה הוּא הוֹצֵאתִי /</p></blockquote>
<blockquote><p>אֲשֶׁר בַּסֵּתֶר עֻשֵּׂיתִי?״ / וַיֹּאמַר ״לֹא חִסַּרְתִּי דָבָר</p></blockquote>
<blockquote><p>מִמִּשְׁאָלֶיךָ / בַּל קְרֹב אֵלֶיךָ, / וְכָל דָּבָר בִּמְחִיר גָּדוֹל</p></blockquote>
<blockquote><p>נִרְצָה / וּבְסֵפֶר זֶה נִמְצָא – / כָּל שֶׁכֵּן בְּחִפָּזוֹן / בַּר</p></blockquote>
<blockquote><p>לֶחֶם וּמָזוֹן! / וְעַתָּה יֶשׁ לְךָ לָתֵת כְּכֹל אֲשֶׁר הוֹצֵאתָ פֹה /</p></blockquote>
<blockquote><p>עִם הַתֶּבֶן וְעִם הַמִּסְפּוֹא – / כֶּסֶף עֲשָׂרָה שְׁקָלִים /</p></blockquote>
<blockquote><p>בְּשֶׁקֶל הָעִיר נִשְׁקָלִים״. / וַיִּשְׁמַע הָאוֹרֵחַ הַדְּבָרִים</p></blockquote>
<blockquote><p>וַיִּתְעַצֵּב / וַיָּקָם עַל פֶּתַח הַמִּשְׁכָּן וַיִּתְיַצֵּב /</p></blockquote>
<blockquote><p>וַיֹּאמֶר בְּלִבּוֹ: וּמַה יּוֹעִיל כַּעְסִי וַחֲמָתִי / וְלָמָה</p></blockquote>
<blockquote><p>אֲחַלֵּל עִמּוֹ צֶדֶק אִמְרָתִי? / וַיֹּאמֶר לוֹ ״קַח אֶת כָּל אֲשֶׁר</p></blockquote>
<blockquote><p>תֹּאמַר, / כִּ יֵין עֲסִיסְךָ לְשׁוֹתָיו יֵמַר״. / וַיִּשְׁקֹל לוֹ</p></blockquote>
<blockquote><p>הַכֶּסֶף כַּאֲשֶׁר הָיָה בוֹחֵר / עוֹבֵר לַסּוֹחֵר.</p> <p>וַיְהִי בְאַשְׁמֹרֶת הַבֹּקֶר וַיַּשְׁכֵּם הָאוֹרֵחַ</p></blockquote>
<blockquote><p>לַעֲמֹס חֲמוֹרוֹ / וְעִמּוֹ נְעָרָיו עוֹשִׂים דְּבָרוֹ. / וַיִּקְרָא אֶל</p></blockquote>
<blockquote><p>הָאִישׁ פֶּתַח הַשַּׁעַר – / וַיָּנַע לְבָבוֹ כְּנוֹעַ עֲצֵי הַיַּעַר /</p></blockquote>
<blockquote><p>וַיָּקָם בְּכֹבֶד וּבַעֲצַלְתַּיִם / כְּשִׁכּוֹר אָסוּר בַּנְחֻשְׁתַּיִם</p></blockquote>
<blockquote><p>/ וַיֹּאמַר ״תֶּן לִי שְׂכַר הַמָּלוֹן וְהַחֲנָיָה / וּשְׂכַר מִטָּה</p></blockquote>
<blockquote><p>יְפֵה־פִיָּה!״ / וַיֹּאמֶר הָאוֹרֵחַ ״כַּמָּה הַמְּחִיר אֲשֶׁר תִּדְרֹשׁ</p></blockquote>
<blockquote><p>/ וּמִפְּנֵי אָדָם לֹא תֵבוֹשׁ?״ / וַיֹּאמַר ״בְּיַד הָאִישׁ הַיְכֹלֶת –</p></blockquote>
<blockquote><p>/ בֶּקַע לַגֻּלְגֹּלֶת!״ / וַיְמַהֵר וַיִּתֶּן לוֹ הַמְּחִיר אֲשֶׁר</p></blockquote>
<blockquote><p>שָׁאַל, / כִּי הוּא מְטֻמָּא וּמְגֹאָל. / וַיִּפְתַּח, וַיֵּלֶךְ</p></blockquote>
<blockquote><p>הָאוֹרֵחַ לִנְתִיבוֹ / וּשְׁנֵי נְעָרָיו סְבִיבוֹ. / וַיֹּאמֶר לָהֶם</p></blockquote>
<blockquote><p>״בָּרְכוּ, בָּנַי, בְּכֹחַ לָאֵל / יְיָ מִמְּקוֹר יִשְׂרָאֵל, / אֲשֶׁר</p></blockquote>
<blockquote><p>הִצִּילָנוּ מִיַד הָאִישׁ הַנָּבָל / אֲשֶׁר הִרְשִׁיעַ לָנוּ וְחָבַל. /</p></blockquote>
<blockquote><p>וְאִם הָיִיתִי יוֹדֵעַ מַעֲלוֹ וְרִשְׁעוֹ / וַהֲבִינוֹתִי נִכְלוֹ</p></blockquote>
<blockquote><p>וּבִצְעוֹ – / לֹא הָיִיתִי לָן בִּמְלוֹנוֹ / וְלֹא קָרַבְתִּי אֶל</p></blockquote>
<blockquote><p>שֻׁלְחָנוֹ. / וְהִנְנִי מִתְנַחֵם לְרֹב פִּשְׁעוֹ וּמַעֲלוֹ / עַל</p></blockquote>
<blockquote><p>הַצֳּרִי אֲשֶׁר נָתַתִּי לוֹ״. / וַיֹּאמְרוּ לוֹ הַנְּעָרִים ״אַל יִחַר</p></blockquote>
<blockquote><p>בְּעֵינֵי יְחִידָתֶךָ / כַּתַר לוֹ זְעֵיר וְתִרְאֶה נִקְמָתֶךָ. / אוּלַי</p></blockquote>
<blockquote><p>הַצֳּרִי יִהְיֶה סִבָּה לַחֲלוֹתוֹ / חַרְצֻבּוֹת לְמוֹתוֹ״. / וַיִּשָּׂא</p></blockquote>
<blockquote><p>הָאִישׁ אֶת עֵינָיו אֶל רוּם מְעוֹנַי / וַיֹּאמַר ״אָמֵן, כֵּן יַעֲשֶׂה</p></blockquote>
<blockquote><p>יְיָ!״ / וַיֵּלֶךְ לְדַרְכּוֹ נִפְלָא וְרַעֶיוֹנָיו תְּמֵהִים – /</p></blockquote>
<blockquote><p>וַיִּפְגְּעוּ בוֹ מַלְאֲכֵי אֱלֹהִים.</p> <p>וְהַכִּילַי קָם מִמִּשְׁכָּבוֹ / וַיֹּאכַל וַיֵּשְּׁתְּ</p></blockquote>
<blockquote><p>וַיִּטַב לִבּוֹ / וַיִּתְהַלֵּל וַיִּשְׁתַּבֵּחַ / עַל כָּל אֲשֶׁר</p></blockquote>
<blockquote><p>עָשָׂה אֶת הָאוֹרֵחַ / וְעַל דְּבַר הַצֳּרִי אֲשֶׁר שָׁאַל לִשְׂכָרוֹ /</p></blockquote>
<blockquote><p>וְלֹא הִרְבָּה בִמְחִירוֹ, / וַיֹּאמֶר בְּלִבּוֹ: יִקְרְבוּ יְמֵי</p></blockquote>
<blockquote><p>הַקַּיִץ וְאֵלֶךְ לִי אֶל הַמְּדִינָה / מְקוֹם הַדַּעַת וְהַבִּינָה /</p></blockquote>
<blockquote><p>וְאֶמְכֹּר אֶת הַצֳּרִי לְאַחַד הַסּוֹחֲרִים / אֲשֶׁר בִּינָה שׁוֹחֲרִים</p></blockquote>
<blockquote><p>/ וְאֶקְנֶה בְדָמָיו מַלְבּוּשִׁים / וְיֵין עֲנוּשִׁים.</p> <p>וַיְהִי מִיָּמִים וְהִנֵּה אִישׁ עוֹבֵר / וְהַצָּמָא</p></blockquote>
<blockquote><p>בְּקִרְבּוֹ גוֹבֵר / וַיָּסַר אֶל בֵּיתוֹ לִשְׁתּוֹת / וַיּוֹאֶל</p></blockquote>
<blockquote><p>הַכִּילַי אוֹתוֹ לְפַתּוֹת / וַיֹּאמֶר לוֹ ״מַה מְּלַאכְתְּךָ וּמֵאַיִן</p></blockquote>
<blockquote><p>תָּבוֹא?״ / לָדַעַת עִסְקוֹ וּמַחֲשָׁבוֹ. / וַיֹּאמֶר לוֹ ״אָנֹכִי אִישׁ</p></blockquote>
<blockquote><p>צַיָּד, / בְּחַרְבִּי וְקַשְׁתִּי צְבִי מִיָּד. / וְהָלַכְתִּי לִקְנוֹת</p></blockquote>
<blockquote><p>לִמְלַאכְתִּי סַם / מֵאֵת אִישׁ אֶחָד בַּשָּׂם, / לָתֵת מִמֶּנּוּ</p></blockquote>
<blockquote><p>בְּרֹאשׁ הַחִצִּים / לִהְיוֹת נוֹגְשִׂים וְאָצִים / לְהָמִית כָּל</p></blockquote>
<blockquote><p>בְּהֵמָה וְחַיָּה / בְּכָל כְּלִי צְדִיָּה. / וְהַסַּם הַזֶּה הוּא חָזָק</p></blockquote>
<blockquote><p>/ וְכָל הַנּוֹגֵעַ בּוֹ יֻזַּק / וְלֹא יוֹעִיל לוֹ מַרְפֵּא וּמִסְעָד, /</p></blockquote>
<blockquote><p>זוּלָתִי הַצֳּרִי הַגָּדוֹל הַבָּא מִגִּלְעָד״. / וַיֹּאמֶר הַכִּילַי</p></blockquote>
<blockquote><p>״תֶּן לִי מְעַט מִמֶּנּוּ וְאָשִׂים מַאֲרָב / אֶל נָחָשׁ שָׂרָף</p></blockquote>
<blockquote><p>וְעַקְרָב / אֲשֵׁר הֵם לוֹחֲצִים אוֹתִי / וְאֶת בְּנֵי תְנוּבָתִי״. /</p></blockquote>
<blockquote><p>וַיִּתֶּן לוֹ מְעַט מִזְעָר / וְחָצְנוֹ נָעַר. / וַיִּפֶן הַצַּיָּד</p></blockquote>
<blockquote><p>וַיָּשָׁב לְבֵיתוֹ / וְהַחַיּוֹת יָגוּרוּ מִשֵּׁתוֹ.</p> <p>וְהַכִּילַי חָשַׁב מַחְשְׁבוֹת אָוֶן בְּתַאֲוָתוֹ, /</p></blockquote>
<blockquote><p>כִּי נִתְמַלְאָה סְאָתוֹ, / וַיֹּאמַר: עַתָּה אֶבְחַן הַצֳּרִי</p></blockquote>
<blockquote><p>וְתִפְאַרְתּוֹ / וְאֵדַע מְחִירוֹ וְתוֹעַלְתּוֹ. / אִם סְגֻלַּת הַסַּם</p></blockquote>
<blockquote><p>וְהַצֳּרִי כַּאֲשֶׁר לָמַדְתִּי יָפֶה – / אֲנִי אֶמָּחֵץ וַאֲנִי</p></blockquote>
<blockquote><p>אֵרָפֵא. / וַיִּקַּח מִן הַסַּם וַיִּתֵּן בִּלְשׁוֹנוֹ / וַיִּלָּכֵד</p></blockquote>
<blockquote><p>בַּעֲווֹנוֹ / וַיִּגְוַע לִבּוֹ כְרֶגַע – / הַמִּקְרֶה וְהַפֶּגַע! – /</p></blockquote>
<blockquote><p>וַיִּתְהַפֵּךְ בְּצִירוֹ וַיִּתְחַלְחַל / וּפָחַד וְרָהָה וְזָחַל. /</p></blockquote>
<blockquote><p>וַיָּרָץ אֶל גְנָזָיו לָקַחַת אֶת הַצֳּרִי / לְהָסִיר אַף וָחֳרִי – /</p></blockquote>
<blockquote><p>וַיִּתְבַּלְבְּלוּ יָדָיו בַּמַּפְתֵּחַ, / אֵין סוֹגֵר וְאֵין פּוֹתֵחַ.</p></blockquote>
<blockquote><p>/ וַיִּשְׁאַג וַיִּנְהֹם בְּמַכְאוֹבוֹ / לְרֹעַ עֲצָתוֹ וּמַחֲשָׁבוֹ. /</p></blockquote>
<blockquote><p>וַתֵּרֶא אִשְׁתּוֹ כִּי נִלְכַּד בְּמַעֲלָלָיו / וְצִירָיו נֶהֶפְכוּ</p></blockquote>
<blockquote><p>עָלָיו, / וַתִּפְתַּח וַתִּקַּח הַצֳּרִי הַשָּׁמוּר כְּבַת עַיִן /</p></blockquote>
<blockquote><p>וַתָּבֵא לוֹ יַיִן – / וְאֵין שׁוֹתֶה וְאֵין לוֹקֵחַ, / אֵין מַבְרִיחַ</p></blockquote>
<blockquote><p>וְאֵין בּוֹרֵחַ! / וַיִּגְוַע וַיָּמָת בַּאֲשָׁמָיו / וְנָפַל</p></blockquote>
<blockquote><p>בַּעֲצוּמָיו, / וְהַמַּרְפֵּא לֹא הוֹעִיל / צָרְיוֹ עִבַּר – וְלֹא</p></blockquote>
<blockquote><p>יַגְעִיל.</p> <br/></blockquote>
